### <a name="azure-active-directory-authentication"></a>Azure Active Directory 驗證

Azure 空間的錨點也可讓應用程式，以向 Azure Active Directory 的使用者權杖。 例如，您可以使用 AAD 權杖，ASA 與整合。 如果企業維持在 AAD 中的使用者，您可以提供 Azure 空間的錨點 SDK 中的使用者 AAD 權杖。 如此一來，可讓您直接與屬於相同的 AAD 租用戶帳戶的 ASA 服務進行驗證。
