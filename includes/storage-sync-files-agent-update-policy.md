Azure 檔案同步代理程式會定期更新，以新增功能和解決問題。 建議您設定 Microsoft Update，以取得 Azure 檔案同步代理程式的最新更新。 我們了解某些組織會想要嚴格控制並測試更新。

若要使用舊版 Azure 檔案同步代理程式進行部署：

- 在最初發行新的主要版本之後，儲存體同步服務會允許使用先前的主要版本三個月。 例如，在發行版本 2.\* 之後，儲存體同步服務會支援版本 1.\* 三個月。
- 過了三個月後，儲存體同步服務會封鎖使用過期版本的已註冊伺服器，讓它們無法與其同步群組同步。
- 在使用舊的主要版本的三個月內，所有 Bug 修正只會套用至目前 (新的) 主要版本。

> [!Note]  
> 如果您使用的 Azure 檔案同步版本在三個月內過期，我們會透過 Azure 入口網站中的快顯通知來通知您。
