## Get SQL server connection information
-->

取得連線到 Azure SQL Database 所需的連線資訊。 在後續程序中，您將需要完整的伺服器名稱、資料庫名稱和登入資訊。

1. 登入 [Azure 入口網站](https://portal.azure.com/)。

1. 從左側功能表中選取 [SQL Database]，然後選取 [SQL 資料庫] 頁面上的資料庫。

1. 在您資料庫的 [概觀] 頁面上，檢閱 [伺服器名稱] 旁邊的完整伺服器名稱。 若要複製伺服器名稱，請將滑鼠暫留在其上，並選取**複製**圖示。  

![server-name](./media/sql-database-connect-query-prerequisites-server-connection-info-includes/server-name.png)

如果您忘記登入資訊，請選取伺服器名稱以開啟 [SQL 伺服器] 頁面。 您可以在此處檢視**伺服器管理員**名稱，然後視需要選取 [重設密碼]。
