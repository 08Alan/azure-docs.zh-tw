對於 Azure Functions 2.x 版中的本機開發，此套件會自動[註冊為繫結擴充功能](../articles/azure-functions/functions-triggers-bindings.md#local-development-azure-functions-core-tools)。 對於 Functions 1.x 和 2.x 中的 C# 類別庫開發，此套件會自動[安裝在專案中](../articles/azure-functions/functions-triggers-bindings.md#local-c-development-using-visual-studio-or-vs-code)。 

請注意，原始程式碼會遵循 WebJobs SDK 版本編號：WebJobs SDK 2.x 版相當於 Azure Functions 1.x，所以 Functions 1.x 程式碼位於存放庫的 v2.x 分支中。 主要和開發人員分支會包含 Web Jobs 3.x 的程式碼，這與 Functions 2.x 相同。
