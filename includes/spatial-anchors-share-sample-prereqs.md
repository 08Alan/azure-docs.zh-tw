## <a name="prerequisites"></a>必要條件

若要完成本教學課程，請確定您具有下列項目︰

* 已安裝 <a href="https://www.visualstudio.com/downloads/" target="_blank">Visual Studio 2017+</a> 且具有 **ASP.NET 和 Web 開發**工作負載的 Windows 機器。
* [.NET Core 2.2 SDK](https://dotnet.microsoft.com/download)。
* 要部署並執行的一或多個裝置 (iOS 或 Android)。
  * 如果使用 Android：
    * 在 Windows 機器上安裝 <a href="https://developer.android.com/studio/" target="_blank">Android Studio 3.3+</a> 和 <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2018.3+</a>。
    * <a href="https://developer.android.com/studio/debug/dev-options" target="_blank">由開發人員啟用</a>且<a href="https://developers.google.com/ar/discover/supported-devices" target="_blank">具備 ARCore 功能</a>的 Android 裝置。
  * 如果使用 iOS：
    * 已安裝 <a href="https://geo.itunes.apple.com/us/app/xcode/id497799835?mt=12" target="_blank">Xcode 10+</a>、<a href="https://cocoapods.org" target="_blank">CocoaPods</a> 和 <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2018.3+</a> 的 macOS 機器。
    * 由開發人員啟用且<a href="https://developer.apple.com/documentation/arkit/verifying_device_support_and_user_permission" target="_blank">與 ARKit 相容</a>的 iOS 裝置。
