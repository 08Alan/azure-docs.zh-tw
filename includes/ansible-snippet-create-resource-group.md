1. 在 Cloud Shell 中建立名稱為 `rg.yml` 的檔案。

    ```bash
    code rg.yml
    ```

1. 將下列程式碼貼到編輯器中：

   ```yaml
   ---
   - hosts: localhost
     connection: local
     tasks:
       - name: Create resource group
         azure_rm_resourcegroup:
           name: ansible-rg
           location: eastus
         register: rg
       - debug:
           var: rg
   ```

1. 儲存檔案並結束編輯器。

1. 使用 `ansible-playbook` 命令執行劇本：

   ```bash
   ansible-playbook rg.yml
   ```
