Azure 当前使用两种部署模型：资源管理器部署模型和经典部署模型。 兩個模型彼此無法完全相容。 開始之前，您必須知道您想要使用哪一個模型。 如需部署模型的資訊，請參閱[了解部署模型](../articles/resource-manager-deployment-model.md)。 如果您不熟悉 Azure，建議使用 Azure Resource Manager 部署模型。
