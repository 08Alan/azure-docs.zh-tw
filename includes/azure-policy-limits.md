沒有每個物件類型的 Azure 原則的最大計數。 _範圍_的項目表示訂用帳戶或[管理群組](../articles/governance/management-groups/overview.md)。

| Where | 何事 | 最大計數 |
|---|---|---|
| 影響範圍 | 原則定義 | 250 |
| 影響範圍 | 計畫定義 | 100 |
| 租用戶 | 計畫定義 | 1,000 |
| 影響範圍 | 原則或方案指派 | 100 |
| 原則定義 | 參數 | 20 |
| 計畫定義 | 原則 | 100 |
| 計畫定義 | 參數 | 100 |
| 原則或方案指派 | 排除項目 (notScopes) | 250 |
| 原則規則 | 巢狀的條件 | 512 |
