


利用 Microsoft HPC Pack 與 Azure 的運算和基礎結構服務，建立及管理雲端高效能運算 (HPC) 叢集。[HPC Pack](https://technet.microsoft.com/library/jj899572.aspx) 是建置在 Microsoft Azure 和 Windows Server 技術上的 Microsoft 免費 HPC 解決方案，並支援各種 HPC 工作負載。雲端 HPC Pack 叢集為叢集系統管理員或獨立軟體廠商 (ISV) 提供富彈性的可調整平台，以執行需要大量運算的應用程式。它也可以協助您降低在內部部署計算叢集上的投資。如需在 Azure 上執行 HPC 和批次工作負載的其他選項，請參閱[批次和高效能運算的技術資源](../articles/batch/big-compute-resources.md)。
