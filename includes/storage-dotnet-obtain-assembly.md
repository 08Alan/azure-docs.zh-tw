### 取得 Storage Client Library 封裝
您可以使用 NuGet 來取得 Storage Client Library 封裝及其相依項目。在 [方案總管] 中以滑鼠右鍵按一下專案，然後選擇 [管理 NuGet 封裝]。在線上搜尋 "WindowsAzure.Storage"，再按一下 [安裝] 以安裝 Azure 儲存體封裝與相依性。

Storage Client Library 封裝也包含在 [適用於 .NET 的 Azure SDK](https://azure.microsoft.com/downloads/) 中。不過，我們建議您從 NuGet 安裝 Storage Client Library，以確保永遠有最新版的用戶端程式庫。
