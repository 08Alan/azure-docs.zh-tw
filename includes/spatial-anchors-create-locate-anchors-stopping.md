## <a name="pause-reset-or-stop-the-session"></a>暫停、 重設，或停止工作階段

若要停止工作階段，您可以叫用了 stop （）。 這樣會停止任何監看員和環境的處理，即使您叫用 ProcessFrame()。 然後，您可以叫用 start （） 以繼續處理。 繼續時，會維護已擷取的工作階段中的環境資料。
