> [!NOTE]
>
> 本文近期已更新為使用新的 Azure PowerShell Az 模組中的命令。 此模組是適用於 Azure 的新 PowerShell 模組，與 PowerShell 5.x 版和第 6 版均相容。 AzureRM 仍會持續取得重大 Bugfix 更新，但只有 Az 模組才會提供新功能。
>
> * 若要深入了解 Az 模組和 AzureRM 模組的持續支援，請參閱[新的 Azure PowerShell Az 模組簡介](/powershell/azure/new-azureps-module-az)。
> * Az 模組已提供於 Cloud Shell 中，並自動啟用與 AzureRM 的相容性。
>   如需安裝 Az 模組的指示，請參閱[安裝 Azure PowerShell](/powershell/azure/install-az-ps)。
> * 若要啟用 AzureRM 相容性命令，請在安裝之後執行 `Enable-AzureRmAlias`。
>   如需相容性的詳細資訊，請參閱[從 Azure PowerShell AzureRM 移轉至 Az](/powershell/azure/migrate-from-azurerm-to-az)。
