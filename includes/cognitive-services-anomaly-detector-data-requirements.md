> [!NOTE]
> 針對使用異常偵測器 API 時獲得最佳結果，應包含 JSON 格式的時間序列資料：
> * 分隔相同的間隔，不能超過 10%的遺漏的點的預期數目的資料點。
> * 如果您的資料沒有清除季節性模式，就會至少 12 個資料點。
> * 至少 4 個模式相符項目如果您的資料沒有清除季節性模式。
