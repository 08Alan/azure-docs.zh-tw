| 資源 | S1 標準 | S2 標準 | F1 免費 |
| -------- | ----------- | ----------- | ------- |
| 裝置/單位 | 500 | 500 | 10 |
| 訊息/天 | 50,000 | 1,500,000 | 3,000 |
| 單位上限 | 200 | 200 | 1 |
| 裝置更新 (建立、更新、 <br/> 刪除) 每天每個單位 | 1100 | 1100 | 1100 |

> [AZURE.NOTE]若您預期單一 S1 或 S2 層的中心會使用超過 200 個單位，請連絡 Microsoft 支援。
