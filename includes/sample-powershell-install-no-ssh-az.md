此範例需要 Azure PowerShell。 執行 `Get-Module -ListAvailable Az` 以尋找版本。 如果您需要安裝或升級，請參閱[安裝 Azure PowerShell 模組](/powershell/azure/install-az-ps)。 

執行 `Connect-AzAccount` 來建立與 Azure 的連線。
