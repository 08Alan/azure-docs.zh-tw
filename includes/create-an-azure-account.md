# 建立 Azure 帳戶並啟用預覽功能

若要將應用程式部署至 Azure，您需要有帳戶。本文說明如何建立免費試用帳戶，以及如何對帳戶啟用 Azure 預覽功能 (包括 BizTalk 服務、HDInsight 和 Azure Back)。

## 建立 Azure 帳戶

1.  開啟網頁瀏覽器，並瀏覽到 <a href="http://www.windowsazure.com">http://www.windowsazure.com</a>。
2.  如果您有 MSDN 訂閱，請瀏覽到 <a href="/pricing/member-offers/msdn-benefits-details/" target="_blank">MSDN 訂戶的 Azure 權益</a>頁面，並依照步驟啟用 MSDN 訂戶的權益來建立帳戶。

   否則，請瀏覽到<a href="/pricing/free-trial/" target="_blank">建立免費試用帳戶</a>頁面，並依照步驟來建立試用帳戶。您將需要有信用卡號和行動電話號碼來證明身分，但是您不會被收取任何費用。

<h2><a id="enable"></a>啟用 Azure 預覽功能</h2>

1.  瀏覽到 [https://account.windowsazure.com/](https://account.windowsazure.com/)，並使用 Azure 帳戶登入。
2.  按一下 [預覽功能]**** 來檢視可用的預覽。<br />
    ![open preview features tab][1]
3.  向下捲動到要啟用的功能，然後按一下 [立即試用]****。<br />
    ![select a preview feature][2]
4.  選取您的訂閱，然後按一下勾號。<br />
    ![select subscription][3]
