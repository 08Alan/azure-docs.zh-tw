此範例需要 Azure PowerShell 模組 5.1.1 版或更新版本。 執行 ` Get-Module -ListAvailable AzureRM` 以尋找版本。 如果您需要安裝或升級，請參閱[安裝 Azure PowerShell 模組](/powershell/azure/install-azurerm-ps)。 

執行 `Connect-AzureRmAccount` 來建立與 Azure 的連線。
