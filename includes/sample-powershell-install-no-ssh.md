此範例需要 Azure PowerShell。 執行 `Get-Module -ListAvailable Az` 以查看其是否已安裝。 如果您需要安裝，請參閱[安裝 Azure PowerShell 模組](/powershell/azure/install-az-ps)。 

執行 `Connect-AzAccount` 來建立與 Azure 的連線。
