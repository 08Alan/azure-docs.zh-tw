#### <a name="to-install-regular-updates-via-the-azure-classic-portal"></a>透過 Azure 傳統入口網站安裝一般更新
1. 在 [裝置]  頁面上，選取要安裝更新的裝置。
2. 瀏覽至 [裝置]  >  [維護]，然後向下捲動至 [軟體更新]。
3. 若要檢查更新，可按一下頁面底部的 [檢查更新]  。
4. 如果有可用的軟體更新，您將會看到一則訊息。 按一下 [安裝更新]  開始更新裝置。
   
    成功安裝更新時，系統將會通知您。
