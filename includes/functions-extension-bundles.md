若要參考 Azure Functions 2.x 預設繫結，請開啟 *host.json* 檔案並更新內容，以符合下列程式碼。

```json
{
    "version": "2.0",
    "extensionBundle": {
        "id": "Microsoft.Azure.Functions.ExtensionBundle",
        "version": "[1.*, 2.0.0)"
    }
}
```
