Azure Marketplace 提供由 Microsoft、協力廠商公司及開放原始碼軟體計劃所開發的各種熱門 Web 應用程式。除了用來連線到 [Azure 入口網站](http://go.microsoft.com/fwlink/?LinkId=529715)的瀏覽器以外，從 Azure Marketplace 建立的 Web 應用程式不需要安裝任何軟體。 

在本教學課程中，您將學到：

- 如何透過 Azure Marketplace 建立新的 Web 應用程式。

- 如何透過 Azure 入口網站部署 Web 應用程式。
 
您將建立使用預設範本的 WordPress 部落格。下圖顯示完成的應用程式：


![Wordpress 部落格][13]

>[AZURE.NOTE] 如果您想要在註冊 Azure 帳戶之前開始使用 Azure App Service，請移至[試用 App Service](http://go.microsoft.com/fwlink/?LinkId=523751)，讓您能夠在 App Service 中立即建立短期的入門 Web 應用程式。不需要信用卡，無需承諾。

## 在入口網站中建立 Web 應用程式

1. 登入 Azure 入口網站。

2. 按一下 [Marketplace] 圖示，或按一下儀表板左下方的 [新增] 圖示，選取 [Web + 行動]，然後選取底部的 [Azure Marketplace] 以開啟 Azure Marketplace。
	
	<!--todo:![Create New][5]-->
	
3. 在 Marketplace 中，選取 [Web 應用程式]。

4. 按一下 [可擴充的 WordPress] 圖示。

	<!--todo:![WordPress from list][7]-->
	
5. 讀取 WordPress 應用程式的描述之後，選取 [建立]。

6. 按一下每個部分 ([Web 應用程式]、[資料庫] 和 [訂閱]****)，然後輸入或選取必填欄位的值。
	
- 輸入您選擇的 URL 名稱	
- 選取最靠近您的區域

	<!--todo:![configure your app][8]-->

7. 定義 Web 應用程式完成之後，按一下 [建立]。

## 啟動與管理 WordPress Web 應用程式
	
1. 從 [Web 應用程式] 分頁中，按一下新的 Web 應用程式，以開啟儀表板。

	<!--todo:![launch dashboard][10]-->

2. 在 [Essentials] 頁面上，按一下 [Url] 底下的連結，以開啟 Web 應用程式的歡迎頁面。

	<!--todo:![site URL][11]--> 

3. 如果您尚未安裝 WordPress，輸入 WordPress 所需的適當組態資訊，然後按一下 [安裝 WordPress] 以完成設定，並開啟 Web 應用程式的登入頁面。

	<!--todo:![login to WordPress][12]-->

4. 按一下 [登入]，然後輸入您的認證。  
	<!--todo:screenshot-->

5. 您的新 WordPress Web 應用程式看起來類似以下的 Web 應用程式。    

	<!--todo:![your WordPress site][13]-->
