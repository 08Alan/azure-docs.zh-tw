**主機**是執行 docker 容器的 x64 型電腦。 這可以您內部部署的電腦，或是在 Azure 中裝載服務的 Docker，這些服務包括：

* [Azure Kubernetes Service](../articles/aks/index.yml)
* [Azure 容器執行個體](../articles/container-instances/index.yml)
* 部署至 [Azure Stack](/azure-stack/operator) 的 [Kubernetes](https://kubernetes.io/) 叢集。 如需詳細資訊，請參閱[將 Kubernetes 部署至 Azure Stack](/azure-stack/user/azure-stack-solution-template-kubernetes-deploy)。
