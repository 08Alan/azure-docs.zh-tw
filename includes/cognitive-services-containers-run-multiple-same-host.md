### <a name="running-multiple-containers-on-the-same-host"></a>在同一主机上运行多个容器

如果您想要使用公開的連接埠執行多個容器，請務必使用不同的公開連接埠執行每個容器。 例如，在端口 5000 上运行第一个容器，在端口 5001 上运行第二个容器。

您可以擁有這個容器和主機上同時執行的不同認知服務容器或您可以執行相同的認知服務容器的多個容器。
