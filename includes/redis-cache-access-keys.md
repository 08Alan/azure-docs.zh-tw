### <a name="retrieve-host-name-ports-and-access-keys-using-the-azure-portal"></a>使用 Azure 入口網站來擷取主機名稱、連接埠及存取金鑰

在連接 Azure Redis 快取執行個體時，快取用戶端需要主機名稱、連接埠和快取金鑰。 某些用戶端可能會以稍有不同的名稱來參考這些項目。 您可以在 Azure 入口網站中擷取這些資訊。

若要使用 [Azure 入口網站](https://portal.azure.com) 擷取存取金鑰，請瀏覽至快取並按一下 [存取金鑰]。 

![Redis 快取金鑰](media/redis-cache-access-keys/redis-cache-keys.png)

若要擷取主機名稱、連接埠，請按一下 [屬性]。

![Redis 快取屬性](media/redis-cache-access-keys/redis-cache-hostname-ports.png)
