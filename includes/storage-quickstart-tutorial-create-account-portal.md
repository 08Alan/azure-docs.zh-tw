## <a name="create-a-storage-account-by-using-the-azure-portal"></a>使用 Azure 入口網站建立儲存體帳戶

首先，建立新的一般用途儲存體帳戶供本快速入門使用。 

1. 使用您的 Azure 帳戶登入 [Azure 入口網站](https://portal.azure.com/#create/Microsoft.StorageAccount-ARM)。 
2. 輸入儲存體帳戶的唯一名稱。 為您的儲存體帳戶命名時，請記住這些規則：
    - 名稱長度必須是 3 到 24 個字元。
    - 名稱僅能包含數字和小寫字母。
4. 選取您的訂用帳戶。 
5. 建立新的 [資源群組] 並提供給它唯一的名稱。 
6. 選取 [位置] 供儲存體帳戶使用。
7. 讓其他欄位設為其預設值。
8. 選取 [釘選到儀表板]，再選取 [建立] 以建立儲存體帳戶。 

建立儲存體帳戶之後，它就會釘選到 Azure 入口網站的儀表板。 選取要開啟的儲存體帳戶。 在 [設定] 底下，選取 [存取金鑰]。 選取主要帳戶存取金鑰，然後按一下 [複製] 按鈕，將相關聯的連接字串複製到剪貼簿。 然後將字串貼上到文字編輯器以供日後使用。
