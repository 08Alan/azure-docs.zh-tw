# <a name="commands"></a>命令
|||
|---|---|
|az mysql server|管理伺服器。 伺服器包含資料庫、資料倉儲和彈性集區。|
|az mysql server create|建立 MySQL 伺服器。|
|az mysql server delete|刪除 MySQL 伺服器。|
|az mysql server list|取得資源群組中所有 MySQL 伺服器的清單。|
|az mysql server show|顯示 MySQL 伺服器的詳細資料。|
|az mysql server update|更新伺服器。|
|az mysql server restore|將伺服器還原至先前的時間點。|
|az mysql server firewall-rule|管理伺服器的防火牆規則。|
|az mysql server firewall-rule create|建立防火牆規則。|
|az mysql server firewall-rule delete|刪除防火牆規則。|
|az mysql server firewall-rule list|列出防火牆規則。|
|az mysql server firewall-rule show|顯示防火牆規則的詳細資料。|
|az mysql server firewall-rule update|更新防火牆規則。|
|az mysql db|管理資料庫。|
|az mysql db list|列出伺服器下所有的 MySQL 資料庫。|
|az mysql server-logs list|列出伺服器的記錄檔。|
|az mysql server-logs download|將記錄檔從伺服器下載至本機。|
|az mysql server configuration list|列出伺服器設定參數。|
|az mysql server configuration show|顯示伺服器設定參數的詳細資料。|
|az mysql server configuration update|修改伺服器設定參數值。|
|||
