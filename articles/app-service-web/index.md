# <a name="azure-web-apps-documentation"></a>Azure Web Apps 文件

App Service Web Apps 是受到完整管理的計算平台，非常適合用來裝載網站和 Web Apps。 Microsoft Azure 的這項平台即服務 (PaaS) 產品可讓您專注於商務邏輯，而讓 Azure 負責處理用來執行及調整應用程式的基礎結構。
