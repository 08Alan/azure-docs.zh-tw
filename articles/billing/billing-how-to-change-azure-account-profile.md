# <a name="change-your-azure-account-profile-information"></a>變更您的 Azure 帳戶設定檔資訊

您的設定檔資訊用來與您連絡您的 Azure 帳戶。 您應該讓它保持更新，讓 Microsoft 可以警示您關於重要通知。

帳戶系統管理員設定檔中的電子郵件地址係供 Microsoft 用來傳送重要的計費、 服務，以及有關帳戶建議相關的通知。 我們建議您指定您在定期檢查電子郵件地址。 帳戶系統管理員，您也可以更新您的設定檔，若要變更帳戶的郵寄地址和電話號碼。

在使用者的設定檔，而不是帳戶管理員的帳戶使用者的電子郵件地址係供 Microsoft 用來傳送服務相關的通知和帳戶的建議。 所有計費的通知會傳送給帳戶管理員。 您也可以更新您的設定檔，若要變更您的郵寄地址和電話號碼，但它不會變更帳戶的郵寄地址或電話號碼。

## <a name="change-your-profile-information"></a>變更您的設定檔資訊

1. 登入 [Azure 帳戶中心](https://account.azure.com/Profile)。
1. 選取 [編輯詳細資料] 按鈕，然後更新 [設定檔] 資訊。

   ![顯示您的 Azure 設定檔資訊的範例](./media/billing-how-to-change-azure-account-profile/profile.png)

### <a name="send-notifications-to-an-additional-email-address"></a>將通知傳送至其他電子郵件地址

您可以讓服務通知和傳送電子郵件地址，除了您的帳戶相關聯的帳戶管理員電子郵件地址的帳戶建議事項。 帳戶系統管理員的電子郵件地址會取得所有訂用帳戶的帳戶，包括計費相關的更新的重要通知。

### <a name="change-the-account-administrator-email-address-in-your-profile"></a>變更帳戶管理員電子郵件地址，在您的設定檔

帳戶管理員是設定 Azure 帳戶以及收到帳戶下所有訂用帳戶相關重要電子郵件通知的人員。 建議您指定帳戶管理員會定期收信的連絡人電子郵件地址。

### <a name="updating-your-profile-email-doesnt-update-your-sign-in-email-address"></a>更新您的設定檔電子郵件不會更新您的登入電子郵件地址

更新設定檔電子郵件地址不會更新您的登入電子郵件地址。 若要變更您的登入電子郵件地址，您必須轉移帳戶的擁有權。

### <a name="updating-your-profile-address-doesnt-update-your-credit-card-billing-address"></a>更新您的設定檔地址不會更新您的信用卡帳單寄送地址

若要了解如何更新帳單資訊，請參閱[變更用來支付 Azure 訂用帳戶的信用卡](billing-how-to-change-credit-card.md)。

### <a name="you-cant-update-your-countryregion"></a>您無法更新您的國家/地區

由於技術限制，我們無法變更現有的帳戶國家/地區。 不過，您可以在想要的國家/地區中建立的帳戶，然後連絡 Azure 支援服務，將服務移轉至該帳戶。

## <a name="service-and-marketing-emails"></a>服務 」 和 「 行銷電子郵件

Microsoft Azure 團隊會傳送服務相關的電子郵件通知您重大警示。 小組也會傳送電子郵件帳戶的建議，也就是行銷電子郵件，以協助您最佳化並調整您的專案。

系統會提示您在[Azure 入口網站](https://portal.azure.com)確認或更新您的電子郵件地址每隔 90 天。 通知的目的是確定 Microsoft 已經更新任何服務相關的緊急通知時的連絡資訊。 Microsoft 會傳送電子郵件給您的 Azure 帳戶相關資訊的設定檔電子郵件地址：

- 服務通知
- 安全性警示
- 計費用途
- 支援
- 行銷通訊
- 最佳做法建議，根據您的 Azure 使用量

輸入您要收到您的帳戶相關的電子郵件地址。 輸入電子郵件地址，您將會是中選擇從 Microsoft 接收的通訊。

![範例顯示更新您的連絡資訊方塊](./media/billing-how-to-change-azure-account-profile/update-contact-information.png)

### <a name="change-your-contact-email-address"></a>變更連絡人電子郵件地址

您可以變更您的連絡人電子郵件地址，使用下列方法：

**選項 1**：登入[Azure 帳戶中心](https://account.azure.com/Profile)。 選取 **編輯詳細資料**，然後更新 設定檔的電子郵件地址資訊。 更新您的設定檔電子郵件地址不會更新您的登入電子郵件地址。

**選項 2**：移至[連絡人資訊](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade)在 Azure 入口網站中的區域，然後輸入電子郵件地址。 您提供的位址會接收服務及行銷通訊。 Azure 帳戶中心的設定檔中的電子郵件地址會更新以使其自動相符。 更新您的設定檔電子郵件地址不會更新您的登入電子郵件地址。

**選項 3**：在  [Azure 入口網站](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade)中，選取與您的姓名縮寫圖示或圖片。 然後，選取操作功能表 (**...**).接下來，選取**我的連絡資訊**從功能表，然後輸入電子郵件地址。 您提供的位址會接收服務及行銷通訊。 電子郵件地址，您的 Azure 帳戶中心設定檔中會更新以使其自動相符。 更新您的設定檔電子郵件地址不會更新您的登入電子郵件地址。

![在 Azure 中顯示的連絡資訊更新的範例](./media/billing-how-to-change-azure-account-profile/azure-contact-information.png)

### <a name="opt-out-of-marketing-emails"></a>選擇退出行銷電子郵件

若要退出接收行銷電子郵件：

1. 移至[要求表單](https://account.microsoft.com/profile/permissions-link-request)提交要求，使用您的設定檔電子郵件地址。 您會收到連結，以更新您的喜好設定的電子郵件。
2. 此連結會顯示您的電子郵件地址是中選擇加入的行銷資訊類型。 清除您想要退出，然後按一下任何主題**儲存**。


![顯示管理通訊的權限選項的範例](./media/billing-how-to-change-azure-account-profile/manage-communication-permissions.png)

### <a name="you-still-receive-service-notifications-if-you-opt-out-of-marketing-communications"></a>如果您選擇退出的行銷通訊仍收到服務通知

如果您提供電子郵件地址進行通訊時，您會收到服務通知，根據您的帳戶。 當您退出，請您只選擇退出的行銷通訊，您仍然收到服務通知。
