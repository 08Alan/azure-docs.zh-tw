# 資料管理：了解 Microsoft Azure 上的 NoSQL

**摘要：**Azure 針對您的 NoSQL 工作負載提供各式各樣的選項，但是判斷所有可用選項之間的差異可能並不容易。本文件說明 Azure 上的 NoSQL 技術，以及它們與管理您資料的關聯式模型在管理上有何差異。

關聯式技術長久以來都是主要的資料處理方法。不過，當您處理大量不同類型的資料時 (現今越來越常見的情況)，便會面臨管理這項資料的新挑戰。關聯式技術有時是最適合管理這項資料的技術，有時則不是。本指南逐步解說這些選項，說明每個選項的用途及為何需要使用它。


**內容：**

* Azure 上的資料：Übersicht	
* Azure 關聯式技術：快速檢閱	
* Azure NoSQL 技術
	* 文件存放區：DocumentDB
	* 索引鍵/值存放區：資料表
	* 資料行系列存放區：HBase
	* 巨量資料分析：HDInsight
