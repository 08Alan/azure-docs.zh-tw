# <a name="supported-browsers-for-azure-iot-central"></a>Azure IoT Central 支援的瀏覽器

*此文章適用於操作員、建置員與系統管理員。*

Azure IoT Central 可透過現代化電腦、平板電腦與瀏覽器來存取。 此文章概述支援的瀏覽器清單。 

建議使用與您的作業系統相容的最新瀏覽器。 支援下列瀏覽器：
- Microsoft Edge (最新版本)
- Safari (最新版本，僅限 Mac)
- Chrome (最新版本)
- Firefox (最新版本)
