# <a name="manage-your-application-preferences"></a>管理您的應用程式偏好設定

*本文適用於操作員、建置員及系統管理員。*

IoT Central 具備可配合您的需求自訂應用程式的彈性。 也會針對個別使用者提供一些彈性，讓您自訂自己的檢視畫面。 本文會介紹各種使用者可套用到設定檔上的自訂選項。

## <a name="changing-language"></a>變更語言

IoT Central 支援 17 種語言。 您可以使用導覽列頂端的**語言選擇器**，切換您偏好的語言。 變更語言之後，我們會記住*您的*選取項目，並將其套用至所有應用程式。 請注意，應用程式中的自訂選項 (例如，儀表板影像) 不會加以當地語系化。 

![IoT Central 語言選擇器](media/howto-manage-profile/language-picker.png)

## <a name="changing-theme"></a>變更佈景主題

可支援深色佈景主題和淺色佈景主題。 當預設佈景主題是深色佈景主題時，在右上方按一下您的名稱，即可切換佈景主題。

![IoT Central 佈景主題選擇器](media/howto-manage-profile/theme-picker.png)
