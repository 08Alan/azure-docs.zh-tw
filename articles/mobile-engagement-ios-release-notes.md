#版本資訊

##2.0.0 (2015/02/17)

-   Azure Mobile Engagement 的最初發行版本
-   連接字串組態取代 appId/sdkKey 組態。
-   已移除從傳送與接收任意 XMPP 實體之任意 XMPP 訊息的 API。
-   已移除傳送與接收裝置之間的訊息的 API。
-   增強安全性。
-   已移除 SmartAd 追蹤。
