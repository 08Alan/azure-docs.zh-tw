# 建立 Azure Active Directory (Azure AD) 租用戶或 Azure AD B2C 租用戶：問題與解決方法

[AZURE.INCLUDE [active-directory-b2c-preview-note](../../includes/active-directory-b2c-preview-note.md)]

## 建立 Azure AD 租用戶

若您第一次無法建立 Azure AD 租用戶，請再試一次。若問題持續發生，請連絡「支援服務」。

## 建立 Azure AD B2C 租用戶 (預覽)

如果您在[建立 Azure AD B2C 租用戶](active-directory-b2c-get-started.md)時遇到問題，請嘗試下列解決方法：

- 若您的租用戶清單未顯示 Azure AD B2C 租用戶，請再試一次。
- 如果 Azure AD B2C 租用戶確實顯示在租用戶清單中，但您收到錯誤訊息，請刪除您剛建立的租用戶並再試一次。例如，您可能會收到下列錯誤訊息：「無法完成建立 B2C 租用戶 'contosob2c'。請瀏覽此[連結](http://go.microsoft.com/fwlink/?LinkID=624192&clcid=0x409)以取得詳細指引。」
- 若這些解決方法無效，請連絡「支援服務」。深入了解[如何提出 Azure AD B2C 支援要求](active-directory-b2c-support.md)。
