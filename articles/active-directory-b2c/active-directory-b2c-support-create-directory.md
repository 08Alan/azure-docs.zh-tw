# <a name="creating-an-azure-active-directory-azure-ad-tenant-or-azure-ad-b2c-tenant-issues-and-resolutions"></a>建立 Azure Active Directory (Azure AD) 租用戶或 Azure AD B2C 租用戶：問題與解決方法
## <a name="creating-an-azure-ad-tenant"></a>建立 Azure AD 租用戶
若您第一次無法建立 Azure AD 租用戶，請再試一次。 若問題持續發生，請連絡「支援服務」。

## <a name="creating-an-azure-ad-b2c-tenant"></a>建立 Azure AD B2C 租用戶
如果您在 [建立 Azure AD B2C 租用戶](active-directory-b2c-get-started.md)時遇到問題，請嘗試下列解決方法：

* 若您的租用戶清單未顯示 Azure AD B2C 租用戶，請再試一次。
* 若您的租用戶清單未顯示 Azure AD B2C 租用戶，而您收到了一個錯誤訊息 (「無法完成 B2C 租用戶 'contosob2c' 的建立。 請瀏覽此 [連結](http://go.microsoft.com/fwlink/?LinkID=624192&clcid=0x409) 以取得詳細指引)，請刪除您剛才建立的租用戶並再試一次。
* 請注意，當您刪除現有的 B2C 租用戶並使用相同的網域名稱加以重建時，會發生已知的問題。 您必須使用不同的網域名稱建立 B2C 租用戶。
* 若這些解決方法無效，請連絡「支援服務」。 深入了解 [如何提出 Azure AD B2C 支援要求](active-directory-b2c-support.md)。
