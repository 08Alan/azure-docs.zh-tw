# <a name="scheduler-documentation"></a>排程器文件

Azure 排程器可讓您以宣告方式描述在雲端中執行的動作。  然後它會排程這些動作並且自動執行。  排程器會維護一個可以檢閱的作業執行結果歷程記錄。
