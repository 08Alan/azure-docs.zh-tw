Azure 資料管理：做出正確決定
============================

**摘要：**Azure 提供了數種儲存和管理資料的方式。此一多樣性讓平台使用者得以解決各種不同的問題。然而，多樣性意味著選擇：哪一個資料管理選項適用於哪一種狀況？

此一簡要概觀將協助您回答這個問題。您可以快速取得 Azure 中各項資料管理技術的摘要：SQL Database、VM 中的 SQL Server、Blob 儲存體和資料表儲存體。接著，請閱讀幾種不同案例的相關資訊，並了解在各種案例下最適合使用前述何種技術。

**目錄：**

-   Azure 資料管理：摘要
-   選擇資料管理技術：範例
    -   建立適用於單一組織的企業或部門應用程式
    -   建立適用於多個組織的 SaaS 應用程式
    -   建立可調整的消費者應用程式
    -   建立巨量資料應用程式
    -   建立資料串流應用程式
    -   將 Windows Server 應用程式移至 Azure
    -   建立使用雲端資料的內部部署應用程式
    -   將備份資料儲存在雲端中
