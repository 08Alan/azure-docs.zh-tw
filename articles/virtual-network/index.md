#<a name="virtual-network-documentation"></a>虛擬網路文件

Azure 虛擬網路 (VNet) 是您的網路在雲端中的邏輯身分。  Vnet 可以完全控制網路與區段的項目至子網路，並且使用它來啟動 Azure 虛擬機器或雲端服務。  此外，您可以使用 Azure 中可用的其中一個連線選項，將 VNet 連線至內部部署網路，將您的網路擴展至雲端，同時保持對資源的控制。
