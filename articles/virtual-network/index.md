# <a name="virtual-network-documentation"></a>虛擬網路文件

了解如何使用虛擬網路 (VNet)，這是您的網路在雲端的邏輯呈現方式。 文件會示範如何設定和管理 VNet、使用它來啟動虛擬機器或雲端服務時、將 VNet 連接到內部部署網路，以及將網路擴展至雲端，同時保有資源的控制權。
