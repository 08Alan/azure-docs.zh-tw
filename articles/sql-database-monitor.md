<h1 id="monitor">如何監視邏輯伺服器和資料庫執行個體</h1>

您在內部部署伺服器上慣用的監視工具和技術 (例如稽核登入、執行追蹤以及使用效能計數器)，並不適用於 SQL Database。在 Azure 上，您可以使用資料管理檢視 (DMV) 來監視資料容量、查詢問題和目前的連線。 

如需詳細資訊，請參閱[使用動態管理檢視監視 SQL Database][]。
