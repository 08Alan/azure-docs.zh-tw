# Azure Marketplace 中的機器學習範例應用程式

Azure Machine Learning 是個新管道，可讓資料科學家像智慧型手機應用程式開發人員一樣發揮其專業知識和創意價值。透過將機器學習服務發佈到 Azure Marketplace，世界上的任何人都可以付費訂閱您新發佈的雲端 API，並在他們的應用程式中使用這些 API。

但是，Azure Marketplace 不只提供基本分析 API。例如，我們已建置一組已完成端對端 ML 應用程式，全程使用 Azure ML 來解決特定的商務需求。不需要資料科學家或 ML 專業人員就可以使用這些 ML 應用程式 - 我們的解決方案中已內建科學。使用者可以提供自己的資料，便可以開始使用。 

例如，這些包括：

* [建議](http://datamarket.azure.com/dataset/amla/recommendations) - 是一個自助式但功能超強的建議引擎，可讓您在網站或電子郵件行銷活動中輕易地新增個人化產品建議。 
  
* [人氣組合](https://datamarket.azure.com/dataset/amla/mba) - 是一個 API，可協助客戶在目錄中找到人氣組合的項目。使用客戶的購買歷程記錄在您的網站中新增「人氣組合」建議，並改善數位存放區中的轉換。

* [異常偵測](https://datamarket.azure.com/dataset/aml_labs/anomalydetection) - 是個 API，可在時間序列資料 (例如伺服器遙測、以感應器為基礎的資料) 中找到異常事件。

這些 API 可讓您使用根據 Azure ML 建置的強大機器學習型服務，您不需要是資料科學家或花許多時間自我學習，您只需要提供自己的資料就可以開始使用這些服務。
