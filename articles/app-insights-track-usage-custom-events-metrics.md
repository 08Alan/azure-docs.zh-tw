# 使用事件和度量來追蹤使用量

### 敬請期待

Application Insights SDK 可讓您在應用程式中插入數行程式碼來監視使用者活動，讓您調整應用程式以符合他們的需求。

我們目前還在將此功能移植到 Microsoft Azure 中的新的 Application Insights 入口網站(如有需要，您可以[在舊版入口網站中查看其運作方式](http://msdn.microsoft.com/library/dn481100.aspx))。

不過，您現在可以立即做的一件事就是[擷取和搜尋以 Trace、NLog 或 Log4Net 產生的診斷事件記錄][diagnostic]。
