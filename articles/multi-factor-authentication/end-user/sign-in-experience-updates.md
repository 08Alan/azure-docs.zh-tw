# <a name="updates-to-the-sign-in-experience-for-online-services"></a>線上服務的登入體驗更新

我們即將更新 Microsoft 線上服務 (如 Office 365 與 Azure) 的登入體驗。 雖然這只是外觀上的變更，您將發現螢幕看起來更為簡潔並直覺化，而更能輕易專注於您所需的工作 — 登入。

## <a name="current-ui-versus-new-ui"></a>現行 UI 與新 UI 的比較

來看看現有的 UI 與新 UI 有何不同。 新的 UI 更新了多項外觀及操作方式，變得較為簡潔並凸顯重點。

![舊的 Microsoft 登入體驗](media/microsoft-sign-in-experience-old.jpg)

![新的 Microsoft 登入體驗](media/microsoft-sign-in-experience-new.jpg)

## <a name="when-will-the-new-ui-appear"></a>新的 UI 將於何時問世？

這項變更的預定時間表如下：

| 時間範圍 | 變更 |
|-----------|--------|
| 五月初 | 您會開始在現有的登入頁面上看到通知橫幅。 您可以從該處了解變更並加以準備。 |
| 五月中旬   | 通知橫幅會改為提供可預覽更新 UI 的連結。 在預覽期間，您仍可使用連結切換回現有的體驗。 |
| 六月中旬  | 更新的登入 UI 可供使用，且所有使用者都會預設為新的 UI 體驗。 |

## <a name="related-info"></a>相關資訊

如果您是系統管理員，您可以根據產品找到更多關於這些更新的資訊。

- **Office 365**。 取得更多關於 [Office 365 訊息中心](https://portal.office.com/adminportal/home#/MessageCenter?id=MC133397&MCLinkSource=DigestMail)更新的詳細資料。 您必須使用系統管理員驗證存取此網站，以進行登入。

- **其他線上服務**。 閱讀此[部落格文章](https://cloudblogs.microsoft.com/enterprisemobility/2018/04/04/upcoming-improvements-to-the-azure-ad-sign-in-experience/)，以取得更多關於更新的詳細資料。
