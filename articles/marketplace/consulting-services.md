#<a name="azure-marketplace-consulting-services"></a>Azure Marketplace 諮詢服務
以連接客戶與服務進而支援並擴展客戶的 Microsoft Azure 使用體驗為目標，Azure Marketplace 可支援列出 Microsoft 的系統整合者 (SI) 合作夥伴所提供的諮詢服務供應項目。 

這些諮詢服務供應項目是由特定客戶使用、有固定的範圍和持續時間、有固定價格或免費，並有已定義的結果。 潛在客戶可以輕鬆地、順暢地要求合作夥伴與其連絡。 合作夥伴會在其 CRM 系統中收到通知，並盡力做到在 48 小時內回應使用者。

架構在 Microsoft Dynamics 365、Microsoft Office 365 和 Microsoft Power BI 上的服務供應項目有資格列入 AppSource。 在此檢閱[這些服務的詳細指導方針](https://smp-cdn-prod.azureedge.net/documents/Microsoft%20AppSource%20Partner%20Listing%20Guidelines.pdf)。 

##<a name="eligible-services-and-partners"></a>合格的服務和合作夥伴

合作夥伴也必須具備其服務相關領域的銀級或金級專長認證。 請參閱以下的合格專長認證：

|專長認證  |解決方案領域  |
|---------|---------|
|雲端平台與基礎結構     |   雲端平台、資料中心      |
|應用程式開發與 ISV     |  應用程式開發、應用程式整合、DevOps       |
|資料管理與分析    |  資料分析、資料平台       |

供應項目必須是以下五種服務類型之一。 請注意，您可以列出不止一種供應項目： 

1. **評估：**可以評估客戶的環境，以判斷解決方案的適用性，並提供估計的成本和時間。 
2. 簡報：可以介紹解決方案或諮詢服務，使用架構、示範和客戶範例引起客戶的興趣。 
3. **實作：**可進行完整安裝以產生完整有效的解決方案。 建議您限制在可於 2 週以內完成實作的解決方案。 
4. **概念證明：**可以判斷解決方案是否會符合客戶需求的有限範圍實作。 
5. **研討會：**可以在客戶場地舉行的互動參與活動，根據客戶的資料或環境而量身打造訓練、簡報、評估或示範。 

除了簡報以外，其他諮詢服務皆可透過親赴現場 (亦即在客戶的場地或合作夥伴的設備舉辦) 或虛擬方式 (亦即透過電話會議、網路會議、遠端實作等方式舉辦) 提供。 簡報必須親赴現場提供。

透過 [Microsoft 合作夥伴網路](https://partner.microsoft.com/en-US/membership/competencies#simple-tab-content-1)深入了解專長認證。

|---------|---------|---------|
|澳大利亞    |德國         |紐西蘭         |
|比利時     |義大利         |挪威         |
|加拿大     |印度         |波蘭         |
|丹麥     |日本         |瑞典         |
|芬蘭     |墨西哥         |英國         |

Azure Marketplace 諮詢服務會先在澳大利亞、加拿大、英國和美國推出。 在合作夥伴提交一定數量且極具吸引力的精選服務供應項目並經 Microsoft 核准後，其他國家/地區的目錄就會推出。

##<a name="publishing-process"></a>發佈程序
想讓您的諮詢服務供應項目出現在 Azure Marketplace 中，需經過以下幾個步驟：

1. **要求：**從[這裡](https://azuremarketplace.microsoft.com/en-us/sell/nominate)傳送您的要求，告訴我們有關貴組織的詳細資訊。 
2. **驗證：**我們會審閱您的要求，驗證您是否符合資格條件。 之後，我們會傳送給您有關列入諮詢服務供應項目的指示。 
3. **提交：**提供指示中所要求的資訊。 所有的內容和支援資料皆必須以英文提交。 
4. **檢閱：**我們的行銷和編輯小組會檢閱您的提交，並加以修改以求最佳的可讀性、文法和效率。 
5. **預備：**之後，您的提交會進入預備階段以供您檢閱。 
6. **發佈：**一旦您和 Microsoft 皆滿意預備階段的內容，您的清單便準備好在 Azure Marketplace 上推出。 清單的推出日期取決於您的供應項目中指定的國家/地區。 如果您選取之國家/地區的目錄已經在 Azure Marketplace 中上線，您的清單就會在幾小時內出現。 否則，等到您選取之國家/地區的目錄一上線，您的清單就會出現。
