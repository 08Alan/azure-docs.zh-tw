# <a name="billing-options"></a>計費選項  

## <a name="commercial-considerations-on-the-marketplace"></a>市集相關的商業考量  
市集不分享下列清單類型的營收。  
*   列出  
*   試用版  
*   使用「自備授權 (BYOL)」計費模型的交易

參與市集店面並不會向您收取額外費用。  
*   如需詳細資訊，請瀏覽「Azure Marketplace 參與政策」頁面：[azure.microsoft.com/support/legal/marketplace/participation-policies](https://azure.microsoft.com/support/legal/marketplace/participation-policies)。  

## <a name="pay-as-you-go-and-bring-your-own-license-billing-options"></a>預付型方案和自備授權計費選項  
使用「預付型方案」計費模型作為發佈選項時，依使用量計費的軟體授權營收會由您與 Microsoft 依 80/20 的比例分成。 單一供應項目可以同時使用「預付型方案」和 BYOL 計費模型來定價。 這兩種計費模型在供應項目層級以不同的 SKU 共存。 您可以在「Cloud Partner 入口網站」上的供應項目中設定此計費模型。 

請思考一下下列範例。  
*   如果您啟用 [預付型方案] 選項，就會有以下結果。  
    | 授權成本 | 每小時美金 $1.00 元 |  
    |:--- |:--- |  
    | Azure 使用量成本 (D1/1 核心) | 每小時美金 $0.14 元 |  
    | *由 Microsoft 向客戶收取的費用* | *每小時美金 $1.14 元* |  

    在此案例中，使用您發佈的 VM 映像時，Microsoft 會收取每小時 $1.14 美元的費用。  
    | Microsoft 收取的費用 | 每小時美金 $1.14 元 |  
    |:--- |:--- |  
    | Microsoft 向您支付授權成本的 80% | 每小時美金 $0.80 元 |  
    | Microsoft 保留授權成本的 20% | 每小時美金 $0.20 元 |  
    | Microsoft 保留 Azure 使用量成本 | 每小時美金 $0.14 元 |  

*   如果您啟用 [BYOL] 選項，則會有以下結果。  
    | 授權成本 | 由您協商和收取的授權費用 |  
    |:--- |:--- |  
    | Azure 使用量成本 (D1/1 核心) | 每小時美金 $0.14 元 |  
    | *由 Microsoft 向客戶收取的費用* | *每小時美金 $0.14 元* |  

    在此案例中，使用您發佈的 VM 映像時，Microsoft 會收取每小時 $0.14 美元的費用。  
    | Microsoft 收取的費用 | 每小時美金 $0.14 元 |  
    |:--- |:--- |  
    | Microsoft 保留 Azure 使用量成本 | 每小時美金 $0.14 元 |  
    | Microsoft 保留授權成本的 0% | 每小時美金 $0.00 元 |  

## <a name="single-billing-and-payment-methods"></a>單一計費和付款方式  
使用「交易」清單選項有一個重要優點，就是會以單一計費方式直接向您的客戶收取授權費用和 Azure 使用量費用。  

在此案例中，Microsoft 會代表您計費及收費。 Microsoft 計費服務可讓您無須建立自己與客戶的採購關係。 單一計費可節省您的時間和資源。 單一計費可協助您將焦點放在經營銷售，而不是收取帳款。 

## <a name="enterprise-agreement"></a>Enterprise 合約  
如果您是 Microsoft 的 Enterprise 合約 (EA) 客戶，便可以使用 EA 來支付 Microsoft 產品費用。 您可以收取產品費用，包括您的 Azure 使用量。 使用 EA 作為付款方式適用於希望軟體和雲端服務的授權年限至少為三年的組織。 您可以分次付款，而不是一次預付款項。 如果您使用「預付型方案」交易清單類型，則您的軟體授權費用會依每季的 EA 超額計費週期計費。  

### <a name="monetary-commitment"></a>承諾用量金額  
如果您是 EA 客戶，便可以將 Azure 新增到您的合約中。 向 Azure 事先承諾用量金額後，即可將 Azure 新增至您的合約。 您的承諾用量金額可供整年使用。 您的承諾用量包括 Azure 中各種服務的任意使用組合。
