# <a name="what-is-appsource"></a>什麼是 AppSource
AppSource 可讓 Microsoft 合作夥伴透過 Microsoft 支援的線上店面，讓客戶找到他們的技術與服務。 身為 AppSource 發行者，您可讓商務使用者參與 AppSource 以尋找、嘗試及取得企業營運 SaaS 應用程式與實作服務，以協助推動商務成果並縮短創造價值的時程： 

| 客戶需求       | AppSource          | 
|:---------------------------------------- |:----------------------------------------------------- |
| **尋找能與其已在使用的 Microsoft 產品搭配的商務解決方案**    | 可讓合作夥伴使用協力廠商應用程式和服務來延伸 Microsoft 的雲端應用程式和技術。 |
| **能夠輕鬆找到適當的解決方案或實作服務。**     | 讓您一次達成探索、試用和取得應用程式和服務、增益集等等。 |
| **產業特定企業營運解決方案，以解決其特定的商業挑戰**   | 提供完善的端對端產業解決方案，以協助滿足許多產業的特定需求。 |
| **應用程式以協助改善生產力、效率與企業洞察力**       | 提供企業營運的應用程式，包括客戶服務、人力資源、營運等。 |
| **有經驗的實作合作夥伴，協助調整應用程式以適應其獨特的情況**     | 為以 Dynamics 365、Power BI、PowerApps 為基礎的解決方案提供諮詢服務供應項目目錄，並在 AppSource 上提供協力廠商應用程式，以協助商務使用者尋找其設計目的是要提供可預測結果的諮詢服務。 |

## <a name="appsource-publishing"></a>AppSource 發佈
透過 AppSource，您可以列出應用程式或諮詢服務，協助客戶在使用 Microsoft 產品 (例如 Office 365、Dynamics 365、Power BI 及 Power Apps) 的同時滿足其需求。 下列供應項目可在 AppSource Marketplace 中取得： 
*   **Dynamics 365 for Finance and Operations**：企業資源規劃 (ERP) 解決方案。 差異化的解決方案是適用於中型至大型公司的 Enterprise Edition 和適用於小型至中型公司的 Business Edition。
*   **Dynamics 365 for Customer Engagement**：客戶關係管理 (CRM) 解決方案，包含 Sales、Customer Service、Field Service 以及 Project Service Automation 應用程式。
*   **Dynamics NAV 受控服務**：Microsoft **Dynamics NAV** 是適用於中型組織的企業資源規劃 (ERP) 軟體套件。 此服務提供適用於製造業、通路業、政府、零售業及其他產業的特殊功能。
*   **Power BI**：商務分析視覺化解決方案。
*   **諮詢供應項目**：由 Microsoft 合作夥伴提供的服務，協助客戶了解、試用及實作特製化技術解決方案。
*   **Cortana Intelligence**：Cortana Intelligence Suite 是由 Azure 提供的獨立，但完整整合的資料與分析平台工具集合。
*   **Office 365**：位於雲端的現代化工作場所。 免費與 Microsoft Word、PowerPoint、Excel 和 OneNote 共同作業。

### <a name="consulting-offers"></a>諮詢供應項目
如果您提供的服務是協助客戶簡化數位轉換或實作解決方案或應用程式，那麼您可以提供此服務，作為在 AppSource 上的諮詢供應項目。 
[檢閱指導方針並了解如何提交您的供應項目](https://smp-cdn-prod.azureedge.net/documents/Microsoft%20AppSource%20Partner%20Listing%20Guidelines.pdf "AppSource - 合作夥伴清單指導方針")。

## <a name="azure-active-directory-integration"></a>Azure Active Directory 整合
需要某些 AppSource 店面應用程式，才能與 Azure Active Directory (Azure AD) 整合並發佈。 與 Azure AD 整合的應用程式已有完善的記載，Microsoft 提供多個 SDK 和額外資源來滿足您的需求。 

| 供應項目名稱 | 需要 Azure AD 整合 | 注意 |
|:--- |:---:|:--- |
| Dynamics 365 for Operations | 否 | 授權型供應項目不需要 Azure AD 整合 |
| Dynamics 365 for Finance | 否 | 授權型供應項目不需要 Azure AD 整合 |
| Dynamics 365 for Customer Engagement | 否 | 授權型供應項目不需要 Azure AD 整合 |
| Dynamics NAV 受控服務 | 否 | 授權型供應項目不需要 Azure AD 整合 |
| Power BI | 否 | 授權型供應項目不需要 Azure AD 整合 |
| 諮詢供應項目 | 否 | 服務是透過合作夥伴協調，而不是透過 Web 體驗 |
| Cortana Intelligence | yes | Azure AD 是確保無縫式客戶經驗及促進最高品質潛在客戶的最佳做法 |
| SaaS 應用程式 (前身為 Web Apps) | 慣用 | Azure AD 是確保無縫式客戶經驗及促進最高品質潛在客戶的最佳做法 |
