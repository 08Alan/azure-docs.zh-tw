# <a name="azure-appsource-and-marketplace-review-policies"></a>Azure AppSource 和 Marketplace 評論原則

Azure AppSource 和 Marketplace 的評等和評論原則會列在此處。

* 在 AppSource 和 Azure Marketplace 評論中，Microsoft 將不會容許下列行為或內容：
    * 聊天機器人/網路攻擊
    * 隱私權合規性違規
    * 垃圾郵件內容
    * 詐騙內容
    * 冒犯的內容：淫穢、不雅的字眼，或攻擊性語言或手勢
    * 不合法的內容
    * 濫用、充滿憎恨，或具威脅性的內容
    * 誹謗性內容
    * 重複送出相同或類似的內容
    * 廣告，包括宣傳其他應用程式和服務

* Microsoft 保留因任何原因而移除使用者提交評論相關內容的權利。
* Microsoft 保留因任何理由阻止使用者提交評論相關內容的權利。
* 發佈者將不會在移除與其應用程式相關的評論相關內容時，收到通知。
* 使用者將不會在移除其提交之評論相關內容時，收到通知。
* Microsoft 沒有義務提供與移除評論相關內容相關的說明。
* Microsoft 將不會接受還原已移除之評論相關內容的要求。

## <a name="review-guidelines-for-customers"></a>適用於客戶的評論指導

* 請不要提交可能會違反 [AppSource 和 Azure Marketplace 評論原則](./rating-review-policies.md)的內容。
* 請不要加入個人識別資訊 (PII)，例如電子郵件地址、實體地址、電話號碼等。
* 請確定您的評論清楚易讀，且具有參考價值。
* 請確定您評論中所表達的觀點平衡且中立。
* 請記住您是在公共論壇上貼文並採取相應行動。

## <a name="review-guidelines-for-publishers"></a>適用於發佈者的評論指導

* 請不要張貼可能會違反 [AppSource 和 Azure Marketplace 評論原則](./rating-review-policies.md)的任何內容。
* 請不要在您的評論中加入任何行銷內容或廣告。
* 請不要在您的評論中宣傳其他應用程式或服務。
* 您的回應應該與特定的應用程式和評論直接相關。
* 請不要使用罐頭回應，而且不要在多個評論中張貼罐頭評論。
* 請不要以任何方式操作評論，例如，對評論者提供變更評等或評論的補償或獎勵。
* 請務必保持禮貌，並考慮滿意的客戶是您應用程式的最佳推銷者。
