# <a name="preferred-solutions-in-azure-marketplace-and-appsource"></a>Azure Marketplace 和 AppSource 中的慣用解決方案

Microsoft 慣用解決方案是指根據品質、效能，以及在特定產業垂直市場或解決方案領域中滿足客戶需求的能力等條件，所選出的雲端應用程式。 由 Microsoft 產業和銷售專家組成的團隊，會從 Microsoft 合作夥伴網路中具有[金級專長認證](https://partner.microsoft.com/membership/competencies)的合作夥伴當中選出解決方案。 這些解決方案會在雲端市集店面中列為精選項目：Azure Marketplace 和 AppSource。 在 AppSource 上，您可以依照產業垂直市場來找尋慣用解決方案；在 Azure Marketplace 上，則可以按一下 [解決方案領域] 中的類別來尋找。

當 Microsoft 將解決方案提供者的一或多項解決方案選為慣用解決方案時，將會連絡提供者。
