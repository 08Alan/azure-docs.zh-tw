# <a name="azure-marketplace-terms"></a>Azure Marketplace 條款 

這些條款將規範 Microsoft Azure Marketplace (以下稱 "Marketplace") 的使用情形，讓您據以存取或購買最適用於 Azure 的產品和服務。 這類產品和服務包括虛擬機器資源庫，以及任何其他 Microsoft Azure 功能。 Marketplace 可能包含 Microsoft 或第三方發行者 (這類發行廠商均簡稱為「發行者」) 所發佈的產品或服務 (「Marketplace 供應項目」或「產品」)。

## <a name="purchasing-and-billing"></a>購買和計費

1. *直接客戶。* 如果您已向 Microsoft 購買 Microsoft Azure 訂用帳戶，下列條款將適用於透過該訂用帳戶購買的 Marketplace 供應項目： 

    1. *付款。* 您同意支付因使用 Marketplace 供應項目而產生的所有相關費用。 這些費用會在使用任何 Microsoft Azure 服務的費用以外另計。 
    1. *價格。* Marketplace 供應項目的定價不包括所有適用的稅金和貨幣兌換結算費用，除非另有說明。 您須獨自負責支付此類稅金或其他費用。 價格可能會按照訂用帳戶類型供應項目的週期性費用訂定，或按照使用類型供應項目的測量單位 (例如執行時數或執行的交易數) 訂定。 
    1. *付款方式。* 當您將 Microsoft Azure 訂用帳戶的付款方法提供給 Microsoft 時，您即已授權 Microsoft 採用該付款方法處理您購買的 Marketplace 供應項目。 例如，信用卡即為付款方法之一。 
    1. *計費。* Microsoft Azure 訂用帳戶合約的計費和付款條款會對您的 Marketplace 購買進行規範。  這些條款適用於隨用隨付或預付型方案的購買。 如果您的購買或使用所產生的費用未在計費期間完成處理，我們可能會在後續的計費期間收取這類費用。 我們會透過 Microsoft Azure 帳戶入口網站為您提供費用清單，您可以在其中檢視和列印費用。 這是我們唯一提供的帳單。 非最終帳單所顯示的任何使用量或支出金額，都只是估計值。  估計值不包括最近的活動，或可能包含根據先前的使用模式所做的預測。
    1. *漲價。* 如果您具有使用中訂用帳戶的 Marketplace 供應項目即將漲價，我們會在漲價前的 30 天預先通知您。 在 30 天的通知期間結束後，若您繼續使用 Marketplace 供應項目，即表示您授權同意以新價格收費。
    1. *退費。* 除非法律另有規定或發行者允許，否則所有費用皆無法退款。
    1. *計費更正。* 如果您發現帳單有誤，您必須在帳單上首次出現錯誤後的 120 天內告訴我們。 若未在該段期間內告知，我們即無須承擔該錯誤所產生的所有責任，也無須受理相關損失的索賠。 我們沒有義務更正錯誤或提供退費。 如果我們發現帳單錯誤，我們將在 90 天內更正該錯誤。

1. *間接客戶。* 如果您向轉銷商購買了 Microsoft Azure 訂用帳戶，您與轉銷商的合約所載的購買和計費條款，將對您透過該訂用帳戶購買的 Marketplace 供應項目進行規範。 

1. *一般。* 無論您向哪個實體購買 Microsoft Azure 訂用帳戶實體，下列條款均適用。 

    1. *預付金。* 除非對 Marketplace 供應項目另有規定，否則 Microsoft 訂用帳戶點數 (例如免費試用、MSDN 或 BizSpark) 或預付金將無法用來購買 Marketplace 供應項目。 此類購買會另外計費。
    1. *保留。* 「保留」是指依據指定的期間和區域預先購買符合條件的 Marketplace 供應項目，例如「軟體保留執行個體」。 保留的購買有指定期間，最多三年。 指定期間結束後，保留就會過期。 未使用的保留不會退款給您。 除非對特定 Marketplace 供應項目另有規定，否則不提供換貨及取消服務。 保留的定價以購買時的定價為準。 軟體的保留執行個體不包含計算的成本。
    1. *自動續約。* 如果您所購買的 Microsoft 訂用帳戶具有自動續約選項，我們或您的轉銷商 (若適用) 將可自動為您的訂用帳戶續約，並可按照任何續約條款向您收取費用。  若要避免這類費用，您可以在續約日期之前取消訂用帳戶。
    1. *免費試用。* 如果您參與了 Marketplace 供應項目的免費試用訂用帳戶，則必須在試用期間結束之前取消訂閱，才能避免產生任何費用。 若您未在免費試用期限結束前取消訂用帳戶，則會依據購買價格向您收取 Marketplace 供應項目的費用。 如果您為免費試用供應項目建立了多個訂用帳戶，適用於所有此類訂用帳戶的試用期間將在您建立第一個訂用帳戶時開始。  所有此類訂用帳戶都將在試用期間結束後轉換為付費供應項目。
    1. *取消。* 若未能按時收到您的全部款項，我們將可暫停或取消您對 Marketplace 供應項目的存取權。 因未付款而暫停或取消存取權，可能會導致您的資料遺失。

## <a name="use-rights"></a>使用權

1. *發行者使用規定。* 您對任何 Marketplace 供應項目的使用權，均受到 Marketplace 供應項目發行者個別提供的使用規定 (以下稱「使用規定」) 所規範。 我們僅對自己發佈的 Marketplace 供應項目負責，而不受任何發行者使用規定的約束。 

1. *BYOL 供應項目。* 如果您已在 Marketplace 以外取得使用特定 Marketplace 供應項目所需的權利，發行者可能會提供此類供應項目 (以下稱「自備授權 (BYOL) 供應項目」)。 如果您使用任何 BYOL 供應項目，則須自行確定您有足夠的權利可使用該供應項目。

1. *Microsoft 軟體產品。* 如果第三方發行者在 Marketplace 中發佈虛擬機器供應項目，且其中包括任何列於 [http://azure.microsoft.com/pricing/details/virtual-machines/](http://azure.microsoft.com/pricing/details/virtual-machines/) 的 Microsoft 軟體產品 (以下稱「Microsoft 軟體產品」)：
    1. Microsoft (而非第三方) 是 Microsoft 軟體產品的授權人；
    1. 除了與第三方供應項目相關聯的任何費用以外，使用 Microsoft 軟體產品還需要支付適用的 Azure 訂用帳戶費用。  轉銷商可以設定這些費用；且
    1. Microsoft 的條款 (而不是第三方的使用規定) 將對供應項目中包含的任何 Microsoft 軟體產品進行使用方面的規範。 這些條款也會規範您在 Microsoft Azure 以外使用 Microsoft 軟體產品的權利 (如果有的話)。

1. *應用程式開發介面。* 我們可能會提供與 Marketplace 搭配使用的應用程式開發介面 (以下稱 "Marketplace API")。 如果您使用任何 Marketplace API 來購買或存取 Marketplace 供應項目，則須負責檢閱及遵守 Marketplace 使用者介面中的相關使用規定。  這些條款可能會隨時變更。 使用任何 Marketplace API 來購買或存取 Marketplace 供應項目，即表示您同意依據付款條款支付所有相關費用。  購買時，您會在 Marketplace 使用者介面中看到付款條款。

## <a name="privacy-and-data-protection"></a>隱私權和資料保護

1. *對發行者揭露的資訊。* 若您購買或使用 Marketplace 供應項目，我們可能會與此類供應項目的發行者分享您的連絡資訊以及交易的詳細資料。 未經您的同意，我們不會與任何發行者分享您的客戶資料。

1. *發行者隱私權原則。* 發行者有責任就其從供應項目收集到的客戶資料或由 Microsoft 提供的客戶資訊，提供隱私權聲明來說明其隱私權措施。 除非另有規定，否則 Microsoft 所發佈的 Marketplace 供應項目，以及 Microsoft 的隱私權、安全性、資料位置和保留原則，均不適用於任何 Marketplace 供應項目。 發行者對任何客戶資料或其他客戶資訊的使用也適用此條款。

1. *個人資料的處理。* Microsoft 是供應項目佈建的相關個人資料的處理方或次要處理方。 Microsoft 向所有客户承諾遵守線上服務條款的歐盟一般資料保護規定條款，自 2018 年 5 月 25 日生效，詳情請見 [http://go.microsoft.com/?linkid=9840733](http://go.microsoft.com/?linkid=9840733)。

## <a name="miscellaneous"></a>其他資訊

1. *金融資料。* 某些 Marketplace 供應項目可能包含金融資料。 依據美國聯邦安全法或其他轄區的安全法，Microsoft 並非代理人/經銷商或註冊的投資顧問單位。  Microsoft 不會向個人提供關於投資、購買或銷售證券或其他金融產品或服務的建議。 Marketplace 中不會包含任何建議或勸誘購買或銷售任何證券的內容。 Microsoft 及其股票報價或指數資料的授權人，皆不會認可或推薦任何特定金融產品或服務。 Marketplace 中的所有內容，包括任何資料集或金融應用程式，皆不會提供專業建議，包括 (但不限於) 投資或稅務方面的建議。

1. *節流。* 為了保護系統，我們可能會限制您對 Marketplace 的要求數目。  Microsoft 也會針對 Marketplace 的使用施加合理的限制。 此外，發行者也可以限制您對其服務的要求數目 (以下稱「特定節流」)。 套用的特定節流限制可能會顯示在發行者的內容詳細資料頁面上。 特定節流限制可隨時變更，且無需另行通知。
