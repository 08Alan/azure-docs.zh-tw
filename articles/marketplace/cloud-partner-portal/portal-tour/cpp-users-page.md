# <a name="users-page"></a>使用者分頁

[**使用者**](https://cloudpartner.azure.com/#users)頁面會顯示 Cloud Partner 入口網站所有已註冊使用者的清單。  資料行包括每個使用者的電子郵件地址、入口網站角色、供應項目類型和發行組織。 結果可以依資料行排序，或針對指定的字串篩選。  

此頁面也會顯示 [新增使用者] 按鈕，用來將使用者新增到您的發行組織。  如需詳細資訊，請參閱[在 Cloud Partner 入口網站上管理使用者](./../../cloud-partner-portal-orig/cloud-partner-portal-manage-users.md)。
