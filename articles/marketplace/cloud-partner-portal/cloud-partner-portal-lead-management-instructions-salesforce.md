# <a name="lead-management-instructions-for-salesforce"></a>Salesforce 潛在客戶管理指示

本文件提供您指示，指導您如何設定 Salesforce 系統，進而讓 Microsoft 提供您業務潛在客戶。 

1. 移到 [設定]。 <br/>
![Salesforce 安裝映像](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce1.png)

2.請在 瀏覽列，移至 建立->自訂->潛在客戶->Web 到潛在客戶。 <br/>
![Salesforce web 到潛在客戶映像](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce2.png)

3.按一下 [建立 Web 到潛在客戶表單] 按鈕。 <br/>
![Salesforce 建立表單映像](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce3.png)

4.在下一頁 (請不要擔心表單中的欄位) 按一下 [產生]。 <br/>
![Salesforce 產生映像](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce4.png)

5.複製 OID，並貼到 Cloud Partner 入口網站的物件識別碼</b><b>欄位。 <br/>
![Salesforce 取得 OID 映像](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce5.png)
