# <a name="saas-application-channel-info-tab"></a>SaaS 應用程式通道資訊索引標籤

使用 [通道資訊] 索引標籤來提供適用於應用程式的通道資訊，包括進入市場 (GTM) 資料和連絡人資訊。

![通道資訊表單](./media/saas-channel-info.png)

## <a name="to-configure-channel-information"></a>設定通道資訊

1. 在[通道資訊] > [GTM 資料] 底下，輸入裝載 GTM 資料的網站 URL。
2. 在 [通道管理員連絡人] 底下，提供一位管理員連絡人，讓 Microsoft 可針對支援和商務問題加以聯繫。 輸入下列欄位的資訊：名稱、電子郵件地址和電話號碼。
3. 選取 [儲存] 以完成通道資訊設定。
