# <a name="saas-application-contacts-tab"></a>SaaS 應用程式連絡人索引標籤

本文說明如何使用 [連絡人] 索引標籤，來識別使用您供應項目之客戶的連絡人。

## <a name="identify-contacts"></a>識別連絡人

使用下一個螢幕擷取畫面作為指南來提供所需的資訊。 以星號標示的所有欄位 (*) 都必須填寫。

![工程和支援連絡人表單](./media/saas-offer-contacts.png)

1. 在 [工程連絡人] 底下，提供下列欄位的資訊：

   - 名稱：輸入您應用程式工程連絡人的姓名。 這名連絡人將收到來自 Microsoft 的技術通訊。
   - 電子郵件：輸入工程連絡人的電子郵件地址。
   - 電話號碼：輸入工程連絡人的電話號碼。 支援 ISO 電話號碼標記法。 如需詳細資訊，請參閱 [E.123](https://en.wikipedia.org/wiki/E.123) \(英文\)。
  
2. 在 [支援連絡人] 底下，提供下列欄位的資訊：

   - 名稱：輸入您應用程式支援連絡人的姓名。 這名連絡人將收到來自 Microsoft 的支援相關通訊。
   - 電子郵件：輸入您應用程式支援連絡人的電子郵件地址。
   - 電話號碼：輸入支援連絡人的電話號碼。 支援 ISO 電話號碼標記法。 如需詳細資訊，請參閱 [E.123](https://en.wikipedia.org/wiki/E.123) \(英文\)。
   - 支援 URL：輸入支援頁面的 URL。

3. 當您完成表單時，選取 [儲存]。
