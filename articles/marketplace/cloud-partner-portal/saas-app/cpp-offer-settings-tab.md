# <a name="saas-application-offer-settings-tab"></a>SaaS 應用程式供應項目索引標籤

本文說明如何設定該供應項目的供應項目設定。

[SaaS App] > [新增供應項目] 頁面開啟後，會出現 [供應項目設定] 索引標籤。 

使用 [供應項目設定] 索引標籤來設定 [供應項目識別]，如下一個螢幕擷取畫面所示。 欄位名稱上附加的星號 (*) 表示此為必填欄位。

![供應項目設定索引標籤](./media/saas-new-offer.png)

## <a name="offer-identity-settings"></a>供應項目識別設定

在 [供應項目識別] 中，您需要提供下表列出的欄位資訊。 必要欄位由星號 (*) 標示。

|    欄位名稱      |    描述    |
|  ---------------   |  ---------------  |
|  **供應項目識別碼\***    |  發行者設定檔內供應項目的唯一識別碼。 此識別碼會顯示在產品的 URL 與計費報告中。 此識別碼只能包含小寫英數字元或連字號 (-)。 此識別碼不能以連字號結尾，且最多不能超過 50 個字元。 請注意，供應項目上架後，此欄位便會鎖住。 例如，若發行者 Contoso 發行供應項目識別碼為 sample-vm 的供應項目，它將會在 Azure Marketplace 中顯示為： https://azuremarketplace.microsoft.com/marketplace/apps/contoso.sample-vm?tab=Overview。                 |
|  **發行者識別碼\***    |  發行者識別碼是您在 Marketplace 中的唯一識別碼。 您的所有供應項目都應該連結到您的發行者識別碼。 儲存供應項目之後，即無法修改發行者識別碼。                |
|  **名稱\***      |   您供應項目的顯示名稱。 此名稱會顯示在 Azure Marketplace 與 Azure 入口網站中。 它最多不能超過 50 個字元。 包含產品的可辨識品牌名稱。 請勿在這裡包含您的公司名稱，除非這是您的行銷方式。 如果您在自己的網站行銷這項供應項目，請確認名稱與您網站的名稱相同。               |
|  |  |

選取 [儲存] 儲存您的進度。
