# <a name="saas-application-publishing-prerequisites"></a>SaaS 應用程式發佈必要條件

本文說明在 Azure Marketplace 上發佈 SaaS 應用程式供應項目的必要條件。 Cloud Partner 入口網站提供對入口網站的角色型存取，可讓多位使用者朝向發佈供應項目的方向共同作業。 如需詳細資訊，請參閱[管理使用者](https://docs.microsoft.com/azure/marketplace/cloud-partner-portal-orig/cloud-partner-portal-manage-users)。

## <a name="publishing-prerequisites"></a>發佈先決條件

若要發佈新的 SaaS 應用程式供應項目，您必須符合下列必要條件：
- 可存取 [Cloud Partner 入口網站](https://cloudpartner.azure.com/)。 如需詳細資訊，請參閱 Azure Marketplace 和 AppSource 發佈指南。
- 代表發行者帳戶發佈供應項目之前，必須有一個具備擁有者角色的人員同意遵守[使用規定](https://azure.microsoft.com/support/legal/website-terms-of-use/)、[Microsoft 隱私權聲明](https://www.microsoft.com/privacystatement/default.aspx)與 [Microsoft Azure 認證方案合約](https://azure.microsoft.com/support/legal/marketplace/certified-program-agreement/)。
- 所有非英文的內容都必須隨附英文語言版本。 此內容包括電子店面文字、文件、螢幕擷取畫面、使用規定及隱私權原則。 提供對非英文版應用程式的實用連結 URL 是可接受的。 
- 備妥中繼資料以供使用。 下列清單 (不完整) 會顯示此中繼資料的範例：
  - 標題
  - 描述 (HTML 格式)
  - 標誌影像 (PNG 格式) 及這些固定的影像大小：40 x 40 像素、90 x 90 像素、115 x 115 像素和 255 x 115 像素。
  - 使用規定及隱私權原則
  - 文件
  - 支援連絡人
