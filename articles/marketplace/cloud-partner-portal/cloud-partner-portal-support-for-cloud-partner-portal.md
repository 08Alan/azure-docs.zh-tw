#<a name="support-for-cloud-partner-portal"></a>支援 Cloud Partner 入口網站

Microsoft 提供各式各樣的產品和服務支援。 務必讓您的查詢能找到正確的支援團隊，對確保適當且及時的回應很重要。 請考慮下列案例，這些案例應有助於將您的查詢傳送給適當的團隊。

-   如果您是透過 Cloud Partner 入口網站的發行者並有來自客戶的問題，請告訴您的客戶透過 [Azure 入口網站](https://portal.azure.com/)的支援連結來要求支援。

-   如果您是透過 Cloud Partner 入口網站的發行者並有與您的應用程式或服務相關的問題，請繼續閱讀以下內容。

本文說明如何針對您所遇到的問題，開啟支援票證。

**1.登入您的 Microsoft 帳戶**

瀏覽至 [Cloud Partner 入口網站](https://cloudpartner.azure.com/)，然後使用公司帳戶或學校帳戶登入。

**2.瀏覽至發行者支援頁面**

從右上方功能表，按一下 [說明] 圖示並選取 [支援] 選項。

![support1][1]

**3.選取問題類型和類別**

![support2][2]

**4.要求支援**

選取問題類型和類別之後，按一下 [提出要求]  按鈕。

**5.提供連絡資訊並描述問題**

儘可能完整地描述問題，並提供任何可能有助於解決問題的檔案 (螢幕擷取畫面、錯誤描述、網路追蹤等)。

![support3][3]

**6.提交確認**

一旦您提交問題後，我們將在 24 小時內確認接收，並指派問題的優先順序和嚴重性。 我們專屬的支援工程師團隊會根據我們的服務層級協定回應時間，協助解決問題。 您可以按照以下螢幕擷取畫面來檢視此**回應時間**。 

![support4][4]

-   如果您需要跟我們談談您的問題，請回覆您針對已開啟的支援案例所收到的電子郵件，我們可以與您約定一些時間。 

-   您可以使用上方螢幕擷取畫面中的**線上檢視事件**連結，隨時檢視您問題的進度。
