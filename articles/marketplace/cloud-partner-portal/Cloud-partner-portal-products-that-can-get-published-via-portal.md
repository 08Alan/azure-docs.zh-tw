# <a name="what-products-can-be-published-via-the-cloud-partner-portal"></a>哪些產品可以透過雲端合作夥伴入口網站發佈

「雲端合作夥伴入口網站」是可供您將 Azure 認證虛擬機器 (VM) 映像發佈到 Microsoft Azure Marketplace 或將您的 Cortana Intelligence/Microsoft R Solutions 應用程式發佈到 Microsoft AppSource 的中心。  

VM 映像是預先設定的映像，包含完整安裝的作業系統與一或多個應用程式。 若要將您的虛擬機器映像發佈到 Azure Marketplace，您必須符合[技術性必要條件](https://docs.microsoft.com/azure/marketplace-publishing/marketplace-publishing-vm-image-creation-prerequisites)才能為您的映像取得認證。
