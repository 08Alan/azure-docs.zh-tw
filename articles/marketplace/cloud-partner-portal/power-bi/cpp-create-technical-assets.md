# <a name="create-power-bi-app-technical-assets"></a>建立 Power BI 應用程式技術資產

對於此供應項目類型，您所需要的主要技術資產將是 Power BI 應用程式，這是主要資料集、報告和 (或) 儀表板、選擇性的連線服務與內嵌資料集的集合。 (Power BI 應用程式是從先前名為[內容套件](https://docs.microsoft.com/power-bi/service-organizational-content-pack-introduction)的資產類型演變而來的。)如需開發這些應用程式的完整資訊，請參閱 [Power BI 應用程式](https://go.microsoft.com/fwlink/?linkid=2028636)。


## <a name="obtaining-an-installation-url"></a>取得安裝 URL

Power BI 應用程式只能在 [Power BI 服務](https://powerbi.microsoft.com/)內建置，且必須使用 [Power BI Pro 授權](https://docs.microsoft.com/power-bi/service-admin-purchasing-power-bi-pro)登入。 您的服務應用程式在 Power BI 中建立並經過測試後，請儲存 Power BI 產生的應用程式安裝 URL。  您必須在 Cloud Partner 入口網站提交表單的[技術資訊索引標籤](./cpp-technical-info-tab.md)中提供此 URL。
 

## <a name="next-steps"></a>後續步驟

在您建立並測試 Power BI 應用程式，並且建立相關聯的供應項目之後，您即可[發佈 Power BI 供應項目](./cpp-publish-offer.md)。
