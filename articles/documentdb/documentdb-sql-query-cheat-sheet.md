# 可下載的 DocumentDB SQL 查詢小密技

**DocumentDB 查詢小密技**中顯示常用的查詢、關鍵字、內建函式和運算子，可協助您快速撰寫 DocumentDB 資料查詢，且其格式適合列印成參考表格。

DocumentDB 支援使用 [SQL](documentdb-sql-query.md) 的關聯式、階層式、和空間式的 JSON 文件查詢，不需要指定結構描述或次要索引。除了標準的 ANSI-SQL 關鍵字和運算子，DocumentDB 也支援 JavaScript 使用者定義函數 (UDF)、JavaScript 運算子和許多的內建函數。

## 下載 DocumentDB 查詢小密技

下載查詢小密技做為快速參考使用，讓您快速撰寫查詢。小密技顯示用來從二個範例 JSON 文件抓取資料的常用 SQL 查詢。為了方便使用，您可以將單面小密列印於信紙大小 (8.5 x 11 英吋) 的紙張上。
