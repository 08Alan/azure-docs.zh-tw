#<a name="documentdb-documentation"></a>DocumentDB 文件

Azure DocumentDB 是完全受管理的 NoSQL 資料庫服務，其建置目的是為了獲得快速且可預測的效能、高可用性、彈性調整、全球發佈以及方便開發。 DocumentDB 是無結構描述的 NoSQL 資料庫，可對 JSON 資料提供豐富且熟悉的 SQL 查詢功能 (一致的低延遲)。
