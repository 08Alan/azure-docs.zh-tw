# 管理 Azure Machine Learning 工作區

您可以藉由管理 Microsoft Azure Machine Learning 工作區 (Azure ML)，來管理存取、部署生產 Web 服務，以及管理 Azure ML Web 服務。此外，儀表板索引標籤會顯示工作區使用情形的概觀，和工作區資訊的快速概覽。

在 **[Web 服務]** 索引標籤中，您可以新增、更新或刪除 Azure ML Web 服務。

## 開始管理工作區頁面

1.  登入您的 Microsoft Azure 帳戶。
2.  在 Microsoft Azure 服務面板中，按一下 **[機器學習]**。
3.  在 **[機器學習]** 清單中，按一下工作區。
