# 為什麼會收到 Azure 訂用帳戶有逾期未付帳款的通知?
如果您是 Azure 訂用帳戶的帳戶管理員，且帳戶沒有準時付款，您會收到一封逾期未付帳款的電子郵件通知，或是在 http://accounts.windowsazure.com 或 http://portal.azure.com 上看到警示。

### 如果您忘記付款會發生什麼事?
服務將取消，您將無法使用資源。服務終止 90 天後，將會刪除任何資料。

### 怎麼解決這個問題?

全額支付您未付的差額。

**案例 1**：如果您使用發票付款模式，將您的付款寄至您的發票底部所列的地址。如需協助，請連絡 [Azure 支援服務](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade)。

**案例 2**：如果帳款未付是因為檔案中登記的信用卡遭拒或已過期，請為訂用帳戶使用其他信用卡或付款方法，或是連絡您的銀行以解決此問題。如需如何變更 Azure 付款方法的指示說明，請參閱[如何變更用來支付 Azure 訂用帳戶的信用卡](./billing-how-to-change-credit-card.md)。您必須以帳戶管理員的身分登入才能進行這項變更。


**案例 3**：如果因為帳戶管理員已離職或變更角色而沒有收到帳單通知，請連絡 [Azure 支援服務](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade) 以解決此問題。
