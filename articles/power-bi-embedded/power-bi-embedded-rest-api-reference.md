# Power BI Embedded REST API

運用 **Microsoft Power BI Embedded**，您可以使用 REST API，透過程式設計方式開發應用程式，以將 **Power BI 報表**整合至您應的用程式。

若要了解 **Power BI Embedded** REST API，請參閱 [Power BI REST 作業](https://msdn.microsoft.com/library/mt711493)。
