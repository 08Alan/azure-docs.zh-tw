# 如何取消 Azure 訂用帳戶

>[AZURE.NOTE]如果您不小心取消訂用帳戶並需要重新啟用，請連絡 [Microsoft 支援](http://go.microsoft.com/fwlink/?linkid=544831&clcid=0x409)尋求協助。

如果您取消訂用帳戶，將會立即刪除您所有的服務。取消前，請務必先備份您要儲存的資料。

1. 以帳戶管理員身分登入 [Azure 帳戶入口網站](https://account.windowsazure.com/subscriptions)。
2. 在 [**按一下訂用帳戶即可檢視詳細資料及使用量**] 下方，選取您要取消的訂用帳戶。</br> ![selectsub](.\media\billing-how-to-cancel-azure-subscription\Selectsub.png)
3. 選取頁面右側的 [**取消訂用帳戶**]。</br> ![cancelsub](.\media\billing-how-to-cancel-azure-subscription\cancelsub.png)
4. 選取 [**是，取消我的訂用帳戶**]。</br> ![cancelbox](.\media\billing-how-to-cancel-azure-subscription\cancelbox.png)
5. 按一下 ![checkbutton](.\media\billing-how-to-cancel-azure-subscription\checkbutton.png)，關閉對話視窗並返回您的訂用帳戶頁面。
