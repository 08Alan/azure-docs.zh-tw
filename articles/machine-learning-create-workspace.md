# 建立 Azure Machine Learning 工作區

Microsoft Azure Machine Learning 工作區 (ML 工作區) 包含在管理及發佈實驗時所需的工具。如需 Machine Learning 工作區的詳細資訊，請參閱連結。

## 建立工作區

1.  登入您的 Microsoft Azure 帳戶。
2.  在 Microsoft Azure 服務面板中，按一下 [機器學習]。

![][]

1.  按一下 [建立 ML 工作區]。
2.  在 [快速建立] 頁面中：
    1.  輸入您的工作區資訊。
        附註： [工作區擁有者] 是您的 Microsoft 帳戶 (<name@outlook.com>) 或組織帳戶。
    2.  按一下 [建立 ML 工作區]。

3.  ML 工作區建立後，您會看見它列示在 [機器學習] 頁面上。
