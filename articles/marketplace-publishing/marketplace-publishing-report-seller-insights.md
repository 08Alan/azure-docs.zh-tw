# 了解賣方 Insights 報告

**什麼是賣方 Insights？**

如果您的供應項目產生使用量，所有 VM 和基於使用量計費的開發人員服務的發行者，將會收到來自 Microsoft 的每月報告。

**我會收到什麼？**

- **歡迎電子郵件：**如果是新的發行者，您會收到一封歡迎電子郵件，通知您將會開始收到賣方 Insights 每月報告。

- **每月銷售報告：**如果您有使用量，您會收到第二封電子郵件，當中附加報告並指示如何存取您的密碼

    - 如果您有 VM 或基於使用量計費的開發人員服務 SKU 產生使用量，您的每月報告將會顯示有關訂單、使用量、市場的詳細資料，以及非 BYOL 供應項目之客戶的詳細資料。
    - 為了保護您的客戶資料，這些檔案會使用密碼鎖住，只有您和 Microsoft 才知道密碼。
    - 如果您的供應項目在當月份沒有產生使用量，我們就「不會」傳送報告。

## 如何閱讀您的賣方 Insights 報告


**依 SKU 和授權類型的訂單：[Marketplace 訂單] 索引標籤**

![readingreportbyorders][2]

- 交叉分析篩選器將會依每個項目來篩選報告。
- 有一個圖表依 Azure 授權類型顯示每月訂單 – 每一顯示該月份依不同 Azure 授權類型的訂單總計。
- 有一個圖表依 SKU 顯示每月訂單 – 每一列顯示所有 SKU 依不同 SKU 的每月訂單總計。
- 有一個圖表顯示依 Azure 授權類型和 Marketplace 授權類型的每月訂單趨勢。
- 有幾個圓形圖顯示依 Azure 授權類型和 Marketplace 授權類型的訂單。
- 有一個資料表顯示依 Marketplace 授權類型、每月總計和所有月份累計總計的每月訂單總計。


**依 SKU 和授權類型的訂單：[Marketplace 使用量] 索引標籤**

![readingreportbyusage][3]

- 交叉分析篩選器將會依每個項目來篩選報告。
- 您要選取標準化的 VM 使用量或未處理的使用量。
- 有一個圖表依 Azure 授權類型顯示每月使用量 – 每一列顯示該月份依不同 Azure 授權類型的使用量總計。
- 有一個圖表依 SKU 顯示每月使用量 – 每一列顯示所有 SKU 依不同 SKU 的每月使用量總計。
- 有一個圖表顯示依 Azure 授權類型和 Marketplace 授權類型的每月使用量趨勢。
- 有幾個圓形圖顯示依 Azure 授權類型和 Marketplace 授權類型的使用量。
- 有一個資料表顯示依 Marketplace 授權類型、每月總計和所有月份累計總計的每月使用量總計。


**[訂單資料] 和 [使用量資料] 索引標籤**

這些索引標籤提供您用來產生報告的詳細資料。

![orderdata][4]

![usagedata][5]



**依 SKU 和授權類型的使用量：[客戶] 索引標籤**

![customerstab][6]

- 請注意機密性條款
- 包含依 SKU 的客戶清單、設定檔資訊、交易日期，以及是否選擇做為促銷連絡人。
- 報告包含依 SKU 的訂單數目和總計。


**法律免責聲明：**

![legal][1]

請仔細閱讀法律免責聲明。如果您有任何問題或意見反應，請按一下免責聲明底部的連結，前往 Marketplace 的支援頁面。

## 如何取得密碼提示？

**瀏覽至 https://publish.windowsazure.com/ 並使用您的 Microsoft 帳戶認證登入** ![passwordreminder][7]

**選取 [發行者] 索引標籤** ![selectpublisherstab][8]


**在 URL 中尋找發行者識別碼** - 使用這個識別碼做為密碼開啟賣方 Insights 的 Excel 檔案。在進一步通知之前，我們將繼續使用這個識別碼做為密碼。- 此外，我們建議您使用 MS Office 2013 搭配 Windows 做為您的活頁簿讀取程式。有些使用者回報使用 MS Office for Mac 有問題。

![publisherid][9]

在 URL 中尋找發行者識別碼 + 使用這個識別碼做為密碼，可開啟賣方 Insights 的 Excel 檔案。在進一步通知之前，我們將繼續使用這個識別碼做為密碼。+ 此外，我們建議您使用 MS Office 2013 搭配 Windows 做為您的活頁簿讀取程式。有些使用者回報使用 MS Office for Mac 有問題。

## 後續步驟  
如果您對報告和 Insights 有任何問題，請連絡我們的支援團隊。

1. 瀏覽至 https://publish.windowsazure.com/ 的支援頁面
2. 選取 [報告和 Insights]
3. 選取 [與報告相關的問題]
4. 按一下 [提出要求] ![sellerinsightsquestions][10]
