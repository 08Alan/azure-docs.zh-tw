什麼是網站？
============

[WACOM.INCLUDE [disclaimer](../includes/disclaimer.md)]

網站是指支援 .NET、Node.js 和 PHP 等常用的 Web 應用程式技術，且無須對現有的應用程式進行任何程式碼變更的 Web 應用程式主機。.NET、Node.js 和 PHP 開發人員用來存取檔案和資料庫等資源的所有一般程式設計模型和資源，在部署至 Azure 的網站上都可繼續正常運作。網站可供使用者存取檔案和資料庫資源，以及現今 Web 應用程式所使用的所有標準組態檔案 (例如 web.config、php.ini 和 config.js)。網站也是執行於雲端的 Web 應用程式的主機，可受到您的控制及管理。
