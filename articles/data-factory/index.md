# <a name="azure-data-factory-documentation"></a>Azure Data Factory 文件

Azure Data Factory 是雲端架構資料整合服務，用來自動移動和轉換資料。 Data Factory 服務建立資料整合方案，以從各種資料存放區擷取資料、轉換/處理資料，並將結果資料發佈至資料存放區。
