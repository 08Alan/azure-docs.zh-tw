# <a name="azure-data-factory-mapping-data-flow-union-transformation"></a>Azure Data Factory 對應資料流程集合聯集轉換

[!INCLUDE [notes](../../includes/data-factory-data-flow-preview.md)]

集合聯集會將多個資料流結合在一起，並使那些資料流的 SQL 集合聯集成為來自集合聯集轉換的新輸出。
