#<a name="dns-documentation"></a>DNS 文件

Azure DNS (網域名稱系統) 是 DNS 網域的服務，採用 Microsoft Azure 基礎結構提供名稱解析。  只要將您的網域裝載於 Azure，就可以像管理其他 Azure 服務一樣，使用相同的認證、API、工具和計費方式來管理 DNS 記錄。
