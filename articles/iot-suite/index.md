# <a name="iot-suite-documentation"></a>IoT 套件文件

即刻開始使用 Microsoft Azure IoT 套件。 使用預先設定的解決方案，並大幅提升開發物聯網 (IoT) 解決方案的速度。 透過使用多個平台 (包括 Linux、Windows 以及即時作業系統) 的裝置 SDK，以新增裝置並連接現有的裝置。 無論專案的大小為何&mdash;皆可輕易地從少數感應器擴充到數百萬個同步連接的裝置，並依賴於 Azure 的整體可用性。
