# Azure IoT 套件的概觀
Microsoft 使用 Azure IoT 服務提供範圍廣泛的功能。這些服務提供企業等級能力，可從裝置收集資料、分析移動中的資料串流、儲存和查詢大型資料集、視覺化即時和歷程記錄資料，並與重要的後端辦公室系統整合。Azure IoT 套件將一組常用服務和擴充的功能封裝在一起，為客戶縮短創造價值的時程。這些擴充功能包括預先設定的解決方案，提供最常見解決方案模式的標準實作。與 IoT 軟體開發套件 (SDK) 結合後，客戶就能夠輕鬆自訂預先設定的解決方案，或利用這些功能做為開發新解決方案的範例。
