# Azure 批次開發程式庫和工具
<p> 取得下列程式庫和工具來開發「Azure 批次」應用程式。


+ [Batch .NET 用戶端程式庫](http://www.nuget.org/packages/Azure.Batch/) (NuGet) - 開發用戶端應用程式以透過 Batch 服務執行工作
+ [Batch .NET 管理程式庫](http://www.nuget.org/packages/Microsoft.Azure.Management.Batch/) (NuGet) - 管理 Batch 帳戶
+ [Batch 總管](https://github.com/Azure/azure-batch-samples/tree/master/CSharp/BatchExplorer) (GitHub) - GUI 應用程式和範例，用以瀏覽、存取和更新 Batch 帳戶內的主要項目，包括作業和工作、計算節點和集區，以及計算節點上的檔案。請參閱[部落格文章](http://blogs.technet.com/b/windowshpc/archive/2015/01/20/azure-batch-explorer-sample-walkthrough.aspx)。
