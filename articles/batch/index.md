#<a name="batch-documentation"></a>Batch 文件

Azure Batch 可讓您在雲端有效地執行大規模的平行和高效能運算 (HPC) 應用程式。 在使用 Batch 服務時，您可以定義用來大規模平行執行應用程式的 Azure 計算資源。 您可以執行隨選或排訂的作業，而不需擔心手動建立或管理基礎結構。
