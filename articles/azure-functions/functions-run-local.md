# <a name="how-to-code-and-test-azure-functions-locally"></a>如何撰寫 Azure 函數並在本機進行測試
> [!NOTE]
> Azure Functions 的本機開發和工具經驗目前為預覽狀態，並且在最後發行版本之前，使用經驗將進行大幅改善。 在本機執行 Azure Functions 主機的最簡單方式是使用 [Azure Functions CLI](https://go.microsoft.com/fwlink/?linkid=832752) 並從 npm 安裝它。 目前只支援 Windows。 即將推出完整文件。 若要針對 Azure Functions CLI 提供意見反應或提出錯誤，請在 [azure-webjobs-sdk-script](https://github.com/Azure/azure-webjobs-sdk-script) GitHub 儲存機制中提出問題。
