#<a name="azure-functions-documentation"></a>Azure Functions 文件

Azure Functions 是可在雲端輕鬆執行程式碼 (「函式」) 片段的解決方案。 您可以撰寫處理手邊問題所需的程式碼，而不需擔心要執行它的整個應用程式或基礎結構。 您也可以您選擇的語言進行開發，例如 C#、F#、Node.js、Python 或 PHP。
