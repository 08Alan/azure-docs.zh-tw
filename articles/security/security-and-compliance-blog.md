# Azure 安全性與法規遵循部落格

[Azure 安全性與法規遵循小組部落格](https://blogs.msdn.microsoft.com/azuresecurity/)是取得 Azure 安全性與法規遵循的最新技術資訊的位置。

此外，請尋找 Azure 安全性與法規遵循小組部落格，以取得 [Azure 資訊安全中心](https://azure.microsoft.com/documentation/services/security-center/)之技術主題的最新更新。

除了技術資訊之外，Azure 安全性與法規遵循小組部落格還會將您指向 Azure 部落格中顯示的 Azure 安全性與法規遵循的最新公告。

請務必檢查 Azure 安全性和法規遵循小組部落格，以查看所有 Azure 安全性相關事項的最新消息。
