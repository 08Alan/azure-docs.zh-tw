# Azure 安全性最佳作法與模式

Azure 提供安全的平台，您可以在其中建立您的解決方案。我們也提供許多服務和技術，可以讓您在 Azure 上建置的服務更安全。因為您可以使用許多選項，所以許多人對於 Microsoft 建議做為改善其 Azure 部署的安全性的最佳作法和模式的項目感到興趣。

我們了解您感到興趣，因此，我們建立了文件集合，說明您可以做的事情、提供正確的內容，我們相信會對於改善 Azure 部署的安全性有幫助。

在這些最佳作法和模式的文章中，我們會針對特定主題討論最佳作法和實用模式的集合。這些最佳作法和模式衍生自我們這些技術的經驗和客戶的經驗。

對於每個最佳作法，我們會說明︰

- 最佳作法是什麼
- 您為何想要啟用該最佳作法
- 如果無法啟用最佳作法，結果可能為何
- 最佳作法的可能替代方案
- 如何學習啟用最佳作法
