# Azure 入口網站鍵盤快速鍵 

## 開啟中樞

| 快速鍵 | 動作 |
|--------|----------|
| N | 開啟通知中樞 |
| S 或 / | 開啟瀏覽 (服務) 中樞 | 
| A | 選取作用中的工作階段 | 
| C | 建立 | 

## 瀏覽

| 快速鍵 | 動作 |
|--------|----------|
| F | 移至第一個開啟的分葉 | 
| H | 切換 Azure 入口網站首頁和目前的分葉 | 
| J | 移至上一個開啟的分葉 | 
| K | 移至下一個開啟的分葉 | 
| L | 移至最後開啟的分葉 | 
| - | 最小化分葉或從最大化還原 | 
| + | 最大化分葉或從最小化還原 | 
| K | 移至下一個開啟的分葉 | 


## 控制
| 快速鍵 | 動作 |
|--------|----------|
| Esc | 關閉中樞功能表、對話方塊、帳戶功能表、任何開啟的功能表 | 
| # | 關閉所有開啟的分葉 | 
| Home | 捲動到目前的分葉的頂端 | 
| End | 捲動到目前的分葉的底部 | 
| CTRL+M | 在編輯控制項內切換控制項之間的 Tab 鍵順序及使用定位點 |
