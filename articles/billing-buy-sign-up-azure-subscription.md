# 如何購買或註冊 Azure 訂用帳戶

本主題將協助您進行下列工作：

* 註冊 Azure 免費試用訂用帳戶
* 將您的 Azure 免費試用訂用帳戶升級為隨用隨付訂用帳戶
* 使用線上服務啟用金鑰 (5x5 金鑰) 啟用您的 Azure 訂用帳戶

## 註冊 Azure 免費試用訂用帳戶
若要註冊 Azure 免費試用訂用帳戶，您必須提供電話號碼、信用卡和 [Microsoft 帳戶](https://www.microsoft.com/zh-TW/account/faq.aspx) (先前稱為 Windows Live ID) 或學校/工作帳戶 (也就是組織 ID)。您的電話號碼和信用卡只限用於身分驗證，不會對信用卡收取費用。

>[AZURE.NOTE]為確定您的支付卡片為有效卡片，我們會要求一筆金額 (在美國為 $1 美元，或所在地區的等值貨幣) 進行授權確認，該筆費用將在 3-5 日內根據財務機構原則退回。您的卡片不會被扣款。

若要註冊 Azure 免費試用訂用帳戶：

1. 開啟 [Azure 免費試用頁面](https://azure.microsoft.com/zh-TW/pricing/free-trial/)，然後選取 [**立即試用**]。

2. 登入您的 Microsoft 帳戶或學校/工作帳戶。

3. 在 [**關於您**] 區段中，輸入您的個人資訊。請注意，一旦指定 [國家/地區] 就無法變更。![個人資訊的畫面顯示](./media/billing-buy-sign-up-azure-subscription/AboutYou.png)

4. [以電話驗證] 會使用您偏好的方式 (撥打電話或簡訊)，將驗證碼傳送至電話。輸入您收到的驗證碼，就能幫助我們驗證您的身分。</br></br> 在 [**以電話驗證**] 區段，選取您國家/地區的國際電話國碼，輸入您的電話號碼 (請勿輸入內線分機號碼)，然後選取 [**傳送簡訊**] 或 [**撥號給我**]。收到驗證碼之後，請在 [**驗證碼**] 方塊中輸入該驗證碼，然後選取 [**驗證碼**] 按鈕確認您的電話號碼。您無法使用 VOIP 電話號碼進行電話驗證。</br> ![有關電話驗證的螢幕擷取畫面](./media/billing-buy-sign-up-azure-subscription/PhoneVerify.png)</br>如果您沒有收到文字訊息或回撥，請嘗試使用不同的電話號碼，或者連絡 [Microsoft 支援](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade)尋求協助。</br>
5. 電話驗證完成之後，請移至 [卡片驗證] 區段。

 ![cardverify](./media/billing-buy-sign-up-azure-subscription/VardVerify.png)</br>

6. 信用卡驗證完成之後，請選取 [**合約**] 選項，然後選取 [**註冊**]。![註冊](./media/billing-buy-sign-up-azure-subscription/Signup.png)</br> 可能需要幾分鐘的時間來設定您的訂用帳戶。如果訂用帳戶未在 15 分鐘內完成設定，請連絡 [Microsoft 支援](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade)尋求協助。</br> 如果出現任何錯誤訊息，請遵循訊息中的步驟來解決問題。常見的錯誤訊息為：

**很抱歉，我們無法完成作業。請稍後再試或重新載入網頁** </br>

在此情況下，請嘗試使用其他瀏覽器完成註冊程序。如果問題仍然發生，請連絡 [Microsoft 支援](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade)尋求協助。

![signuperror1](./media/billing-buy-sign-up-azure-subscription/signuperror1.png)</br>7.訂用帳戶就緒時，請選取 [**開始管理我的服務**]。![startservice](./media/billing-buy-sign-up-azure-subscription/startservice.png)

## 將您的 Azure 免費試用訂用帳戶升級為隨用隨付訂用帳戶

1. 以您用來註冊免費試用的 Microsoft 帳戶或學校/工作帳戶登入 [Azure 帳戶入口網站](https://account.windowsazure.com/subscriptions)。
2. 選取 [訂用帳戶]，然後選取 [**升級**]。</br>![billpage](./media/billing-buy-sign-up-azure-subscription/billpage.png)
3. 選取 [**是**]，然後選取 [**立即升級**]。</br>![升級](./media/billing-buy-sign-up-azure-subscription/Upgrade.png)

升級至隨用隨付訂用帳戶之後，您可以繼續使用試用權益 (30 天/$200 美元)，直到天數或金額其中之一用完為止。

## 使用線上服務啟用金鑰 (5x5 金鑰) 啟用您的 Azure 訂用帳戶

如果您向轉銷商購買 Azure in Open 信用額度，請遵循下列步驟啟用您的訂用帳戶：

1. 開啟 [[Azure in Open 授權金鑰](http://azure.microsoft.com/offers/ms-azr-0111p/)] 頁面，然後選取啟用新的訂用帳戶或現有的訂用帳戶。
2. 輸入產品金鑰，然後核取第一個合約選項 (如果您想要收到特殊優惠訊息，亦可核取第二個選項) 以啟用訂用帳戶。</br>![OSA 金鑰](./media/billing-buy-sign-up-azure-subscription/OSAkey.png)
3. 選取 [**啟用**]。
