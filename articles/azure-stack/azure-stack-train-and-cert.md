# <a name="azure-stack-training-and-certification"></a>Azure Stack 訓練與認證

*適用範圍：Azure Stack 整合系統*

下列訓練與認證機會可協助您了解 Azure Stack，並展現您的 Azure 熟練度。

## <a name="training"></a>訓練

- 隨選 Microsoft 官方課程
   - [ODX20537:Configuring and Operating a Hybrid Cloud with Microsoft Azure Stack (180 Day)](https://www.microsoft.com/en-us/learning/course.aspx?cid=ODX20537) (使用 Microsoft Azure Stack 設定和操作混合式雲端 (180 天))
   - [OD20537:Configuring and Operating a Hybrid Cloud with Microsoft Azure Stack (90 Day)](https://www.microsoft.com/en-us/learning/course.aspx?cid=OD20537) (使用 Microsoft Azure Stack 設定和操作混合式雲端 (90 天))

- Microsoft IT 訓練課程
   - [Course 20537A:Course 20537A: Configuring and Operating a Hybrid Cloud with Microsoft Azure Stack](https://aka.ms/azsmoc) (課程 20537A：使用 Microsoft Azure Stack 設定和操作混合式雲端)

- Open edx
   - [edX:Configuring and Operating Microsoft Azure Stack online course](https://aka.ms/AzureStackMOOC) (edX：設定和操作 Microsoft Azure Stack 線上課程)
   
- Microsoft 學習路徑
   - [工作角色與學習路徑](https://azure.microsoft.com/training/learning-paths/)

## <a name="certification"></a>認證

*Configuring and Operating a Hybrid Cloud with Microsoft Azure Stack certification* (使用 Microsoft Azure Stack 設定和操作混合式雲端認證)，[測驗 70-537](https://www.microsoft.com/learning/exam-70-537.aspx)
