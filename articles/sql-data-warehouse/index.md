# <a name="sql-data-warehouse-documentation"></a>SQL 資料倉儲文件

Azure SQL 資料倉儲結合 SQL Server 關聯式資料庫與 Azure 雲端相應放大功能。  SQL 資料倉儲是以我們的巨量平行處理 (MPP) 架構為基礎，可以處理您的企業工作負載。
