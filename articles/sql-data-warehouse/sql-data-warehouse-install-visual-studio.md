# 安裝 Visual Studio 2015

若要進行開發，Azure SQL 資料倉儲團隊建議使用 Visual Studio 2013 或更新版本搭配 SQL Server Data Tools (SSDT)。以下內容將概述如果您尚未安裝可行的 Visual Studio 版本，要如何下載和更新 Visual Studio 2015。下列步驟可協助您下載並安裝與 Azure SQL 資料倉儲搭配使用的必要項目。

+ [下載並安裝 Visual Studio 2015](#Download-and-Install-Visual-Studio-2015)
+ [下載並安裝 SQL Server Data Tools (SSDT) Preview] (#Download-and-Install-SQL-Server-Data-Tools-(SSDT)-Preview)
 
> [AZURE.NOTE]Visual Studio 2013 與 SQL Server Data Tools (SSDT) Preview 也可以和 Azure SQL 資料倉儲搭配使用。
 
##下載並安裝 Visual Studio 2015

若要安裝 Visual Studio 2015，請依照下列步驟

1. 從 Visual Studio Online [下載 Visual Studio 2015](https://www.visualstudio.com/downloads)。
2. 請選擇預設組態，依照 MSDN 的[安裝 Visual Studio](https://msdn.microsoft.com/library/e2h7fzkw.aspx) 指南 

##下載並安裝 SQL Server Data Tools (SSDT) Preview

Azure SQL 資料倉儲會針對 Visual Studio 2013 或 2015，

1. 使用 [下載 SQL Server Data Tools Preview](https://msdn.microsoft.com/mt429383.aspx) 中可用的 SQL Server Data Tools (SSDT) 最新預覽版本。
2. 依照安裝指示來安裝資料工具。
