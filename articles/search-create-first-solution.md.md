# 使用 Azure 搜尋建立您的第一個搜尋解決方案

## 敬請期待

感謝您的耐心等候，我們正在對示範 Azure 搜尋功能的範例應用程式進行收尾工作。幾天後應該就能提供給您。當一切準備就緒，您就可以下載範例程式碼並依循指示，在您的 Azure 搜尋服務中執行它。

有幾個步驟今天就能完成。若想提早開始，可先下載一些項目並設定服務：

-   Visual Studio 2013 Express for the Web
-   設定 Azure 搜尋預覽

### 下載 Visual Studio 2013 Express for the Web

範例應用程式是透過 Visual Studio 2013 所編寫。如果您已有現成的 Visual Studio 2013 版本，請加以使用。否則，可以從 Microsoft 下載免費版本。

前往[下載中心][]以取得 Visual Studio 2013 Express for the Web。

### 設定免費搜尋服務

建議您使用此教學課程的免費服務。這是「搜尋」的限制版本，免費提供給所有訂戶。如果您沒有訂用帳戶，可以註冊試用版。

如需設定服務的詳細資訊，請前往[在 Azure 預覽入口網站上設定搜尋][]。

指示中的步驟比您實際需要的還多。建立好免費服務後，就可以停止步驟。您將會需要服務 URL 和 API 金鑰來完成此教學課程。
