# 資料管理：了解 Azure 上的 NoSQL 技術

**摘要：**Azure 提供許多選項來處理 NoSQL 工作負載，但可能不容易決定所有可用選項之間的差異。本文件說明 Azure 上的 NoSQL 技術，以及從管理您資料的關聯式模型中管理的差異。

關聯式技術長久以來都是主要的資料處理方法。不過，當您使用不同類型的大量資料時 (這種情況逐漸增多)，您會面臨管理此資料的許多新挑戰。關聯式技術有時最適合管理此資料，有時卻不然。本指南逐步解說這些選項，說明每個選項的用途及為何需要使用它。


**目錄：**

* Azure 上的資料：Übersicht	
* Azure 關聯式技術：快速檢閱	
* Azure NoSQL 技術
	* 文件存放區：DocumentDB
	* 索引鍵/值存放區：資料表
	* 資料欄系列存放區：HBase
	* 巨量資料分析：Oozie 協調器
