# SQL Database 的用戶端開發和快速入門程式碼範例


本主題提供可用來連接到 Azure SQL Database 之快速入門程式碼範例的連結。這些範例涵蓋：


- 各種程式設計語言。
- 您的用戶端程式碼執行所在的 Windows 和 Linux 作業系統。
- 簡短的快速入門程式碼範例，以及處理自動重試邏輯之暫時性錯誤的較長範例。
- 將關聯式結果集轉換成物件導向格式的程式碼範例。
