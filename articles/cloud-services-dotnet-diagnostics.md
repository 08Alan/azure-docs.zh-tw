啟用 Azure 診斷
===============

Azure 診斷可讓您從 Azure 中執行的背景工作角色或 Web 角色收集診斷資料。診斷資料可用來進行偵錯和疑難排解、測量效能、監視資源使用量、流量分析和容量規劃以及稽核。

在 Visual Studio 2012 或 Visual Studio 2013 內，您可在部署之前或在執行階段使用 Azure SDK 2.0 或更新版本來設定診斷。如需詳細資訊，請參閱[設定 Azure 診斷](http://msdn.microsoft.com/zh-tw/library/windowsazure/dn186185.aspx)。

如需建立記錄及追蹤策略，以及使用診斷和其他技術來疑難排解問題的其他深入指引，請參閱[開發 Azure 應用程式的疑難排解最佳作法](http://msdn.microsoft.com/zh-tw/library/windowsazure/hh771389.aspx) (英文)。

如需在應用程式手動設定診斷，或遠端變更診斷監視器組態的相關資訊，請參閱[使用 Windows Azure 診斷收集記錄資料](http://msdn.microsoft.com/zh-tw/library/windowsazure/gg433048.aspx)。
