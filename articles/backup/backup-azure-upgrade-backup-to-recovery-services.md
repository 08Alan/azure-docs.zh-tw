# <a name="upgrade-a-backup-vault-to-a-recovery-services-vault"></a>將備份保存庫升級至復原服務保存庫

本文說明如何將備份保存庫升級至復原服務保存庫。 升級程序不會影響任何執行中的備份作業，且不會遺失備份資料。 將備份保存庫升級至復原服務保存庫的主要理由：
- 備份保存庫的所有功能都會保留在復原服務保存庫中。
- 復原服務保存庫較備份保存庫具有更多功能，包括更完善的安全性、整合式監控、更快速還原和項目等級還原、管理改良與簡化入口網站的備份項目。
-  只會發行復原服務保存庫的新功能。

## <a name="how-to-upgrade"></a>如何升級？

將備份保存庫升級至復原服務保存庫的需求很大。 為了簡化此程序，我們會分批將客戶釋出到升級佇列中。 若要註冊升級佇列中的項目，請使用適當的連結：
- [升級備份保存庫](https://www.surveymonkey.com/r/Y57BJQX)
- [升級 Site Recovery 保存庫](https://www.surveymonkey.com/r/5HHPZQN)

## <a name="what-happens-after-i-sign-up"></a>註冊後會發生什麼事？
一旦您的訂用帳戶列入允許清單等候升級，Microsoft 會連絡您繼續進行升級。 如需升級程序的詳細資訊，請參閱 Azure 備份[部落格](http://azure.microsoft.com/blog/upgrade-classic-backup-and-siterecovery-vault-to-arm-recovery-services-vault)。
