#<a name="backup-documentation"></a>備份文件

Azure 備份是一項雲端服務，您可用來備份 (或保護) 和還原 Microsoft Cloud 中的資料。  Azure 備份將以一個可靠、安全及具成本競爭力的雲端架構解決方案，取代您現有的內部部署或異地備份解決方案。
