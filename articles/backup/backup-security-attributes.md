# <a name="security-attributes-for-azure-backup"></a>Azure 備份的安全性屬性

Azure 服務的各個層面都整合了安全性。 本文說明 Azure 備份內建的安全性屬性。 安全性屬性是促使 Azure 服務能夠防止、偵測或因應安全性弱點的一種服務特性或功能。

安全性屬性可分類為︰
* 預防
* 網路分割
* 偵測
* 身分識別與存取管理支援
* 稽核線索
* 存取控制 (如有使用)
* 組態管理 (如有使用)

我們可在每個類別中確認是否使用了某個屬性 (是/否)。 一個屬性有可能不適用於某些服務，而顯示為 N/A。 此外也可提供關於屬性的注意事項或詳細資訊連結。
