#如何登入執行 Windows Server 的虛擬機器#

若要登入執行 Windows Server 的虛擬機器，請使用管理入口網站中的 [連接] 按鈕啟動遠端桌面連線。(對於 Linux VM，請參閱[如何登入執行 Linux 的虛擬機器](./virtual-machines-linux-how-to-log-on/).)
