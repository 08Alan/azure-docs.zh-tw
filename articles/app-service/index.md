#<a name="app-service-documentation"></a>App Service 文件

App Service 是 Microsoft Azure 的平台即服務 (PaaS) 產品。 建立適用任何平台或裝置的 Web 與行動應用程式。 整合您的應用程式與 SaaS 解決方案、與內部部署應用程式連接，並使您的商務程序自動化。
