# <a name="app-service-on-linux-documentation"></a>Linux 上的 App Service 文件
Linux 上的 App Service 目前處於公開預覽狀態，可讓客戶在 Linux 平台上以原生方式執行 Web 應用程式。 對於特定種類的應用程式，這樣可提高應用程式的相容性，輕鬆就能將 Linux 平台上裝載於其他地方的現有 Web 應用程式，移轉到 Azure App Service。

客戶只有在已啟用 Linux 的 App Service 方案內才能建立 Linux 上的 Web 應用程式，而且此 App Service 方案只能裝載以 Linux 為基礎的 Web 應用程式。 

> Linux 上的 App Service 公開預覽價格為正常價格的 50%，此價格是將 App Service 方案所耗用的計算時數減半來計算。
> 
> 公開預覽期間不提供 SLA。
