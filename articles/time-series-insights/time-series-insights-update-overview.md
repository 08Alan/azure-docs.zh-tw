# <a name="azure-time-series-insights-overview"></a>Azure 時間序列深入解析概觀

Azure 時間序列深入解析 (TSI) 是端對端的平台即服務，可用來內嵌、處理、儲存和查詢與內容高度相關且已經過時間序列最佳化的 IoT 規模資料。 因此，Azure TSI 十分適用於特定資料探索及營運分析。 TSI 是唯一可延伸、可自訂且符合產業 IoT 部署廣泛需求的服務供應項目。

## <a name="what-is-iot-data"></a>什麼是 IoT 資料？

IoT 資料是指資產密集組織中提供的所有「產業」資料。 IoT 資料的非結構化程度通常很高，因為這些資料來自各種資產，而這些資產記錄了很多各式各樣的量測，例如溫度、動作和溼度等。 此外，這些資料流通常有差距大、訊息損毀和讀數錯誤的特點。 因此，必須先整理這些資料流中的資料，才能執行任何分析。 通常，IoT 資料只有在搭配來自第一方 (例如 CRM 或 ERP) 或第三方資料來源的其他資料輸入 (例如天氣或位置) 時才有意義。

因此，該資料中只有極少部分可用於營運和商務用途。 這類資料會提供一致、完整、最新且正確的資訊，以用於商務報表和分析。 因此，將收集的 IoT 資料轉換成可付諸行動的深入解析需要幾個重要功能：

* 資料處理功能，用來清理、篩選、插入、轉換及準備資料，以進行分析
* 組織結構功能，用來瀏覽和了解資料 (將資料標準化與情境化)
* 符合成本效益的儲存體，用來長期/無限期保留可用數十年的已處理 (衍生) 資料及未經處理資料

## <a name="azure-time-series-insights-for-industrial-iot"></a>用於產業 IoT 的 Azure 時間序列深入解析

目前的 IoT 環境十分多元。 包含的客戶橫跨製造業、汽車業、石油與天然氣、能源與公用事業、智慧建築和顧問事業等。 案例包括特定資料探索 (資料型態未知)，以及透過結構描述化 (明確模型化) 的資料進行營運分析，以提升營運效率。 這些案例通常是並存的，而且支援不同使用案例。 多層式儲存體 (經常性與非經常性存取) 等平台功能、儲存數十年可用時間序列資料的能力，以及明確模型化和最佳化資產型營運智慧查詢的能力，都將成為產業 IoT 企業和其數位革命的成功關鍵。

Azure TSI 是全方位的端對端平台即服務供應項目，適用於 IoT 資料探索及營運深入解析。 時間序列深入解析提供完全受控雲端服務來分析 IoT 規模的時間序列資料。

客戶可以將未經處理資料儲存在無結構描述的記憶體內存放區，並透過分散式的查詢引擎和 API 執行特定互動式查詢，以及運用我們豐富的使用者體驗，在幾秒內視覺化數十億個事件。 深入了解我們的[資料探索功能](./time-series-insights-overview.md)。

TSI 也會提供目前處於公開預覽的營運智慧功能。 結合互動式資料探索和營運智慧功能，時間序列深入解析可讓客戶從 IoT 資產收集的資料衍生出更多價值。 具體而言，公開預覽供應項目支援下列主要功能：

* 可調整且已最佳化效能與成本的時間序列資料存放區，用於讓雲端式 IoT 解決方案在幾秒內完成多年時間序列資料的趨勢。
* 支援語意模型，用於描述與衍生和非衍生訊號 (來自資產和裝置) 相關聯的網域和中繼資料。
* 結合資產型資料深入解析的強化使用者體驗，其包含用來推動商務及營運智慧的各種特定資料分析
* 整合進階機器學習服務和分析工具，例如 Azure Databricks、Apache Spark、Azure Machine Learning、Jupyter Notebook 和 Power BI 等，以協助客戶解決時間序列資料的挑戰，並提升營運效率。

營運深入解析與資料探索會隨附在適用於資料處理、儲存和查詢的隨用隨付定價模型，進而提供客戶更具延展性的模型，以符合他們的商務變動需求。

在引入這些主要的產業 IoT 功能之後，Azure TSI 提供下列主要優點。

* 適用於 IoT 規模時間序列資料的多層式儲存體

  * 透過用於擷取資料的常見資料處理管線，客戶可以將資料儲存在經常存取的儲存體以進行互動式查詢，和/或使用非經常性存取儲存體來儲存大量資料，以利用高效能的資產式查詢。

  * 近期將推出儲存層之間的動態路由。

* 用於情境化原始遙測資料和衍生資產型深入解析的時間序列模型

  * 客戶可透過具描述性的時間序列模型來情境化原始遙測資料，並透過效能與成本高度最佳化的裝置型查詢，來衍生豐富的營運智慧。

* 與其他資料解決方案無縫整合
  
  * 由於 Azure 時間序列深入解析中的資料存放在開放原始碼的 Apache Parquet 檔案中，因此客戶可以輕鬆地整合其他資料解決方案 (第一方或第三方) 來處理端對端案例，包括營運智慧、進階機器學習服務和預測性分析等。

* 近乎即時的資料探索

  * 藉由 Azure 時間序列深入解析總管的使用者體驗，您可以為透過擷取管線串流的所有資料提供視覺效果。 連線至事件來源後不久，客戶即可檢視、探索和查詢事件資料，這些資料可用於驗證裝置是否如預期般發出資料，以及監視 IoT 資產的健康情況、生產力和整體效益。

* 根本原因分析和異常偵測

  * Azure 時間序列深入解析總管有多個模式和透視檢視，可支援進行和儲存多步驟的根本原因分析。 透過與 Azure 串流分析結合，客戶可使用時間序列深入解析來近乎即時地偵測警示與異常狀況。

* 在時間序列深入解析平台上建置自訂應用程式

  * Azure 時間序列深入解析支援 JavaScript SDK，其具有豐富的控制項和簡化的查詢存取，可讓客戶在時間序列深入解析平台上方建置自訂 IoT 應用程式，以符合個人商務需求。
  * 客戶也可以直接使用時間序列深入解析查詢 API，將資料導到自訂 IoT 應用程式中。
