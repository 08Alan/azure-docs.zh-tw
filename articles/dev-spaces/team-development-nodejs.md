### <a name="make-a-code-change"></a>進行程式碼變更
請移至 `mywebapi` 的 VS Code 視窗，並對 `server.js` 中的預設 GET `/` 處理常式進行程式碼編輯，例如：

```javascript
app.get('/', function (req, res) {
    res.send('mywebapi now says something new');
});
```
