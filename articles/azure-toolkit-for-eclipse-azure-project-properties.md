# <a name="azure-project-properties"></a>Azure 專案屬性
您可以在適用於 Eclipse 的 Azure 工具組內設定 Azure 專案的各種設定。

## <a name="configuring-azure-project-properties"></a>設定 Azure 專案屬性
透過 Azure 專案的屬性對話方塊，即可完成設定 Azure 專案的屬性。 在 Eclipse 的 [專案總管] 窗格中，在專案上按一下滑鼠右鍵，按一下 [屬性]，然後選取 [Azure] 子功能表。

![][ic719480]

在此屬性頁面中，您可以變更下列項目： 

* **服務名稱：** 您的 Azure 專案的名稱。
* **建置目的：** 該專案是為了計算模擬器或部署至 Azure而建置。
* **目標 OS：** 專案部署至 Azure 後，要使用的作業系統。
