# Azure 資料目錄的鍵盤快速鍵

## 一般的鍵盤快速鍵

下列為 Azure 資料目錄的鍵盤快速鍵。

|動作|按
|---|---
|把焦點依序放在頁面的每個控制選項上。|Tab 鍵
|選擇某個選項。|Enter 鍵或空格鍵
|將頁面最大化。|WIN 鍵 + 向上鍵
|將頁面還原/最小化 | WIN 鍵 + 向下鍵
|開啟頁面功能表| ALT 鍵 + 空格鍵


## 驗證頁面

|動作|按
|---|---
|登入服務。|ALT 鍵 + S

## 資料來源選取頁面

|動作|按
|---|---
|前往下一頁 (如果已啟用)。|ALT 鍵 + N
|當焦點在某個磚上時，變更已選取的類型。|向左鍵、向上鍵、向右鍵、向下鍵

## 資料來源連線頁面

|動作|按
|---|---
|前往上一頁 (如果已啟用)。|ALT 鍵 + P
|選擇 [連線] 按鈕。| ALT 鍵 + C

## 連線內容頁面

|動作|按
|---|---
|[註冊] 按鈕 (如果已啟用)。| ALT 鍵 + R
|包括 [預覽] 核取方塊 (如果有提供)。|ALT 鍵 + P
|包括 [資料設定檔] 核取方塊 (如果有提供)。|ALT 鍵 + D
|瀏覽至每個清單項目。|向上鍵或向下鍵
| 在伺服器階層中展開樹狀檢視的節點。 |向右鍵
| 針對 [可用物件] 或 [要註冊的物件] 來移動已選取項目。 | 空格鍵

## 發佈進度頁面

|動作|按
|---|---
|[取消] 按鈕|ALT 鍵 + C
|[註冊更多物件] 按鈕| ALT 鍵 + R
|[檢視入口網站] 按鈕 | ALT+V

## 發佈進度頁面

|動作|按
|---|---
|[儲存] 按鈕 (如果已啟用)。| ALT 鍵 + S
|[取消] 按鈕|ALT 鍵 + C
