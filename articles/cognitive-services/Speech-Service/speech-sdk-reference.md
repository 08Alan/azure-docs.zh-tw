# <a name="speech-sdk-reference"></a>語音 SDK 參考

語音軟體開發套件 (SDK) 提供您應用程式對語音服務功能的存取權，以更輕鬆地開發啟用語音的軟體。 目前，SDK 提供對**語音轉換文字**、**語音翻譯**和**意圖辨識**的存取權。
