# <a name="speech-sdk-api-reference"></a>語音 SDK API 參考

「語音軟體開發套件」(SDK) 可讓您的應用程式存取「語音服務」的功能，使其能夠更輕鬆地開發支援語音的軟體。 目前，SDK 提供對**語音轉換文字**、**語音翻譯**和**意圖辨識**的存取權。 如需有關功能和所支援平台的一般概觀，請參閱文件[入口頁面](http://aka.ms/csspeech)。
