# <a name="speech-sdk-reference"></a>語音 SDK 參考

語音軟體開發套件 (SDK) 提供您應用程式對語音服務功能的存取權，以更輕鬆地開發啟用語音的軟體。 目前，SDK 可供存取**語音轉文字**和**語音轉譯**。 文字轉換語音採用 HTTP 的 [REST] (https://docs.microsoft.com/azure/cognitive-services/speech-service/rest-apis) POST 呼叫。

下表說明目前支援的程式設計語言和作業系統。

|程式設計語言|平台|API 參考|
|-|-|-|
|C#|Windows|[Browse](https://aka.ms/csspeech/csharpref)
|C/C++|Windows、Linux|[Browse](https://aka.ms/csspeech/cppref)
|JAVA\*|裝置|[Browse](https://aka.ms/csspeech/javaref)

\* *JAVA SDK 是[語音裝置 SDK](speech-devices-sdk.md) 的一部分。*
