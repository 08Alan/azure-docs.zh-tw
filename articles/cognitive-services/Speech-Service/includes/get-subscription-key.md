> [!NOTE]
> 請先取得訂用帳戶金鑰。
> 您可以從[認知服務訂用帳戶](https://azure.microsoft.com/try/cognitive-services/)頁面，取得免費試用訂用帳戶金鑰。 您選取語音 API 之後，請選擇 [取得 API 金鑰] 取得兩個金鑰 (主要和次要)。 您可以使用任一金鑰。 這兩個金鑰繫結至相同的配額。
>
> 付費訂用帳戶金鑰皆可透過 Azure 儀表板取得。
