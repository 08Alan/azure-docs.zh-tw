    <h1>文字分析 API 文件</h1>
    <div class="intro" style="min-width: 200px">
        <p>文字分析 API 是一種雲端服務，針對原始的文字提供進階的自然語言處理，同時包含三種主要功能：情緒分析、關鍵片語擷取以及語言偵測。 透過我們的快速入門、教學課程及範例，了解如何以不同的方式分新內容。</p>
    </div>
