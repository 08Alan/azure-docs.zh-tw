# <a name="view-dashboard"></a>檢視儀表板

按一下 [儀表板] 索引標籤，針對在此工具內進行的所有內容審查，查看其重要計量。 您可以查看影像和文字的總計、已完成和進行中審查。 您可以查看預設小組以及子小組的計量。
