# <a name="sign-up-for-content-moderator"></a>設定 Content Moderator

登入，以在內容仲裁[審核工具](https://contentmoderator.cognitive.microsoft.com)網站。 您可以使用您的 Microsoft 帳戶註冊，或在網站上建立帳戶。

## <a name="name-your-team"></a>命名您的小組

為您的審查小組提供名稱。 (選擇性) 您可以邀請同事加入您的小組。
