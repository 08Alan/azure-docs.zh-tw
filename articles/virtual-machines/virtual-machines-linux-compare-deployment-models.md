# Azure Resource Manager 部署模型提供之 Linux 應用程式的 Azure 計算、網路和儲存提供者
Azure Resource Manager 部署模型包含的計算、網路與儲存功能，可以大幅簡化各種 IaaS 複雜應用程式的部署和管理。很多應用程式需要各種的資源組合，其中包括虛擬網路、儲存體帳戶、虛擬機器以及網路介面。Azure Resource Manager 部署模型可以建構 JSON 範本，然後作為單一應用程式部署所有這些資源，並集中管理。
