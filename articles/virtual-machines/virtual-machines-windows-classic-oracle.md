# 在 Azure 中的 Windows 虛擬機器上託管的 Oracle 軟體


設定要在 Azure 基礎結構服務中的 Windows 虛擬機器上執行之第一個或下一個開發/測試或實際執行的 Oracle 資料庫。利用簡化設定的優點以及快速擴展 Oracle 資料庫工作負載的功能，來使其增加新的容量。您也可以在 [Linux 虛擬機器](virtual-machines-linux-classic-oracle.md)上設定 Oracle 軟體。
