# <a name="azure-cosmos-db-whitepapers"></a>Azure Cosmos DB 白皮書 

白皮書可讓您探索更深層的 Azure Cosmos DB 概念。 本文提供 Azure Cosmos DB 的可用白皮書清單。

| **白皮書** | **說明** |
| --- | --- |
| [NoSQL 資料庫雲端服務的擁有權總成本](https://aka.ms/cosmosdb-tco-paper) |本文件會比較在不同的部署組態 (例如內部部署資料庫、使用雲端架構的資料庫等) 中，擁有和操作軟體的財務成本總計。 本文件也會針對高、中、低使用量類型來提供擁有權總成本預估 |
|[使用 Azure Cosmos DB 編製不限結構描述的索引](http://www.vldb.org/pvldb/vol8/p1668-shukla.pdf) | 本文件會說明 Azure Cosmos DB 的編制索引子系統。 本文件包含 Azure Cosmos DB 功能，例如文件表示法、查詢語言、文件索引編制方法、核心索引支援，以及早期的生產經驗。|
| [Azure Cosmos DB 和 GDPR](https://servicetrust.microsoft.com/ViewPage/TrustDocuments?command=Download&downloadType=Document&downloadId=87cc6456-4b23-473c-94d3-6c713b8b8956&docTab=6d000410-c9e9-11e7-9a91-892aae8839ad_FAQ_and_White_Papers)| 本文會為管理雲端式資料庫和 (或) 內部部署資料庫的 Azure Cosmos DB 客戶提供指引，也會為需要確保其資料庫系統中的個人資料處理及保護，是否符合目前規則的人員提供指引。 |
