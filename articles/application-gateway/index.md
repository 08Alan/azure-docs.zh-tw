#<a name="application-gateway-documentation"></a>應用程式閘道文件

Azure 應用程式閘道提供應用程式傳遞控制站 (ADC) 服務。  它會將 CPU 密集 SSL 終止卸載至應用程式閘道，讓客戶最佳化 Web 伺服陣列的產能。  它是您的應用程式專用的虛擬應用裝置，具有延展性和高可用性，以及一組豐富的診斷和記錄以取得較佳管理性。
