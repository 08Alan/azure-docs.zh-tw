# <a name="azure-portal-keyboard-shortcuts"></a>Azure 入口網站鍵盤快速鍵
## <a name="open-hubs"></a>開啟中樞
| 快速鍵 | 動作 |
| --- | --- |
| N |通知 |
| / |瀏覽服務 |
| A |使用中工作階段 |
| B |計費 |
| C |建立 |

## <a name="navigation"></a>瀏覽
| 快速鍵 | 動作 |
| --- | --- |
| H |Azure 啟動儀表板 |
| F |第一次開啟刀鋒視窗 |
| J |上一個開啟的刀鋒視窗 |
| K |下一次開啟的刀鋒視窗 |
| L |上次開啟的刀鋒視窗 |
| - |最小化分葉或從最大化還原 |
| + |最大化分葉或從最小化還原 |
| # |關閉目前的刀鋒視窗 |
| ¦ |關閉所有刀鋒視窗 |
| ， |將焦點移至命令列 |
| 。 |將焦點移至導覽列 |
| ? |檢視鍵盤快速鍵 |
