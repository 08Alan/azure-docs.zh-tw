# <a name="your-debit-card-or-credit-card-is-declined-at-azure-sign-up"></a>在 Azure 註冊時信用卡或金融卡遭到拒絕
註冊 Azure 時，如果您的金融卡或信用卡遭到拒絕或未被接受，則可能有下列問題中的一個需要修正。

* **您的國家/地區不接受信用卡或金融卡提供者。** 選擇信用卡時，只會看到在您所選國家/地區中有效的選項。 如果您的信用卡具備國際交易功能，請連絡您的銀行或發卡機構來進行確認。 請參閱 [支援的國家/地區與貨幣](billing-countries-and-currencies.md)。
* **您的信用卡或金融卡資訊不正確或不完整**。 您輸入的姓名、地址以及 CVV 碼必須完全符合卡片上列印的內容。
* **您使用的是虛擬卡或預付卡。** 虛擬卡、預付信用卡或金融卡不被視為 Azure 訂用帳戶的付款方式。
* **智慧卡非作用中或已被鎖卡。** 請連絡您的銀行，並確定卡片作用中且未被鎖卡。
* **您嘗試啟動第二個免費試用版。** 每位使用者只能啟動一次 Azure 免費試用優惠。 如果您曾利用此信用卡啟動免費試用版，或擁有已連結至此信用卡的 Azure 訂用帳戶，即無法啟動免費試用版。 使用來自 [Microsoft Azure 優惠詳細資料](https://azure.microsoft.com/support/legal/offer-details/)的另一個 Azure 方案。 

更新完付款資訊後，請嘗試再次註冊。

## <a name="business-that-doesnt-want-to-pay-by-card-set-up-invoicing"></a>不想要使用信用卡支付的企業？ 設定發票
如果您代表企業，也可以透過開立發票的付款方法來支付 Azure 訂用帳戶費用，如支票、隔夜支票及電匯。 設定好按發票付款之後，即無法變更為另一個付款選項。 若要按發票付款，請參閱 [Azure 帳單 - 如何開立發票](https://azure.microsoft.com/pricing/invoicing/)。

## <a name="update-your-credit-card-or-debit-card-information"></a>更新您的信用卡或金融卡資訊
註冊之後，[管理您的付款資訊](billing-how-to-change-credit-card.md)以變更或移除卡片。 

## <a name="need-help-contact-support"></a>需要協助嗎？ 請連絡支援人員。
如果仍需要協助，請[連絡支援人員](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade)以快速解決您的問題。
