# IoT 中心 SDK

本文提供各種 [Microsoft Azure IoT SDK][] 的資訊，以及其他資源的連結。

## IoT 中心裝置 SDK

Microsoft Azure IoT 裝置 SDK 包含有助於建置裝置和應用程式的程式碼，並使裝置和應用程式連接到 Azure IoT 中心服務，且由其進行管理。

您可從 GitHub 下載下列 IoT 裝置 SDK：

- 為了可攜性和廣泛平台相容性，以 ANSI C (C99) 撰寫之 [Azure IoT 裝置 SDK for C][]。
- [Azure IoT 裝置 SDK for .NET][]
- [Azure IoT 裝置 SDK for Java][]
- [Azure IoT 裝置 SDK for Node.js][]

### 作業系統平台與硬體相容性

這份[文件][OS Platforms and hardware compatibility]說明裝置 SDK 與不同作業系統平台的相容性，以及 [IoT 程式的 Microsoft Azure 認證][]中包含的特定裝置組態。

## IoT 中心服務 SDK

Microsoft Azure IoT 服務 SDK 包含有助於建置應用程式的程式碼，並使該應用程式直接與 IoT 中心互動，以便管理裝置和安全性。

您可從 GitHub 下載下列 IoT 服務 SDK：

- [Azure IoT 服務 SDK for Node.js][]
