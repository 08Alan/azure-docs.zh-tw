#<a name="azure-government-documentation"></a>Azure Government 文件

Microsoft Azure Government 提供的雲端平台以安全性、隱私權和控制、法規遵循和透明度的基本準則為建置基礎。 公共部門實體會獲得實體隔離的 Microsoft Azure 執行個體，其針對採用其架構建置的所有系統和應用程式，運用了對美國政府至關重要的世界級安全性和法規遵循服務。
