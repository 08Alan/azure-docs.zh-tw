# <a name="justice-and-public-safety-jps-in-azure-government"></a>Azure Government 中的司法與公共安全 (JPS)
## <a name="overview"></a>Overview
司法與公共安全 (JPS) 機關正承受巨大壓力，需要保持社區安全、減少犯罪並改善回應能力。 從智慧型警務感知系統、全國的隨身錄影系統，到日常的行動警務合作，雲端運算正在改變執法機關的工作方式。

若得到適當規劃和保護，雲端服務可以提供 JPS 強大的新功能。 這些功能包括數位證據管理、資料分析和即時決策支援，並在行動裝置上提供最新的解決方案。 不過，並非所有的雲端提供者都可相提並論。 執法機關若要欣然接受雲端，必須要有足以信賴的雲端服務提供者。 執法任務的關鍵，需要合作夥伴致力於滿足各種安全性、法規遵循和作業上的需求。

從裝置到雲端，Microsoft 將隱私權和資訊安全性放在首位，同時提高現場執法人員和整個部門人員的產能。 藉由結合高度安全的行動裝置，以及「隨時隨地」存取雲端的能力，JPS 機關可以參與正在進行的調查、分析資料、管理物證，並協助市民免受威脅。

其他雲端提供者只將美國刑事司法資訊系統 (CJIS) 的法規遵循視為核取方塊，而不是一項承諾。 在 Microsoft，我們致力於提供不論是現在還是未來，都能符合適用 CJIS 控管的解決方案。 此外，透過<a href="http://news.microsoft.com/presskits/dcu/#sm.0000eqdq0pxj4ex3u272bevclb0uc#KwSv0iLdMkJerFly.97">數位犯罪防治單位</a>、<a href="https://channel9.msdn.com/Blogs/Taste-of-Premier/Satya-Nadella-on-Cybersecurity">網路防禦營運中心</a>和<a href="https://enterprise.microsoft.com/en-us/industries/government/public-safety/">全球司法與公共安全組織</a>，我們持續致力於司法與公共安全。
