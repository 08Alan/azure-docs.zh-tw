# <a name="azure-marketplace-for-government"></a>政府用 Azure Marketplace
Azure Marketplace 具有市集發行者所提供，已更新的映像清單，可供 Azure Government 使用。 

## <a name="variations"></a>變化
下面是使用政府用 Azure Marketplace 時，需要考量的事項：

* 只提供自備授權 (BYOL) 映像。 您無法存取任何需要透過 Azure Marketplace 購買交易的映像
* 相較於公開市集，目前這裡只提供映像子集。 您可以在[這裡](../azure-government-image-gallery.md)找到可用的映像清單 
* 您的企業系統管理員在佈建映像之前，必須啟用 Azure 訂用帳戶的 Marketplace 購買項目
  * 請以企業系統管理員身分登入入口網站
  * 瀏覽到 [管理]
  * 在 [註冊詳細資料] 底下，按一下 *Azure Marketplace* 明細項目旁的鉛筆圖示
  * 視需要切換 [啟用/停用]
  * 按一下 [儲存] 

> [!NOTE]
> 如果想要讓您的映像可供 Azure Government 使用，請參閱[合作夥伴上架指導方針](documentation-government-manage-marketplace-partners.md)以取得詳細資訊。
>
