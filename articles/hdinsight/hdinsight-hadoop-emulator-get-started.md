# 透過虛擬機器的 Hadoop 沙箱開始使用 Hadoop 生態系統

您可以在虛擬機器上從 Hortonworks 設定 Hadoop 沙箱，以了解 Hadoop 生態系統。沙箱提供本機開發環境，讓您了解 Hadoop、Hadoop 分散式檔案系統 (HDFS)，以及作業提交。

若要開始使用 Hortonworks Hadoop 沙箱，請參閱 [Hortonworks 沙箱](http://hortonworks.com/downloads/#sandbox)，並查閱 **Hortonworks Sandbox on a VM (VM 上的 Hortonworks 沙箱)**區段。建議您以沙箱操作一遍教學課程，以了解 HDFS、作業如何提交至叢集、如何追蹤叢集上執行的工作等等。

熟悉 Hadoop 之後，您就可以開始在 Azure 中使用 Hadoop 建立 HDInsight 叢集。有關如何開始使用的詳細資訊，請參閱[開始在 HDInsight 中使用 Hadoop](hdinsight-hadoop-linux-tutorial-get-started.md)。
