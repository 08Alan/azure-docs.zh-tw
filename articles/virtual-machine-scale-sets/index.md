#<a name="virtual-machine-scale-sets-documentation"></a>虛擬機器擴展集文件

虛擬機器擴展集是一個您可以用來部署和管理一組相同 VM 的 Azure 計算資源。  所有的 VM 設定相同，VM 擴展集是設計來支援 true 的自動調整 - 不需要預先佈建 VM。 這可讓您更輕鬆地針對大量計算、巨量資料和容器化工作負載建置大規模服務。
