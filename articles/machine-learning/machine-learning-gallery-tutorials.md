# <a name="discover-and-share-tutorials-in-the-cortana-intelligence-gallery"></a>探討與分享 Cortana Intelligence Gallery 中的教學課程
[!INCLUDE [machine-learning-gallery-item-selector](../../includes/machine-learning-gallery-item-selector.md)]

## <a name="tutorials"></a>教學課程
有一些**[教學課程](https://gallery.cortanaintelligence.com/tutorials)**可引導您了解機器學習的技術與概念，或是說明可解決各種機器學習問題的進階方法。

## <a name="discover"></a>探索
若要瀏覽 Gallery 中的教學課程，請開啟 [Gallery](http://gallery.cortanaintelligence.com)，將滑鼠指向 Gallery 首頁頂端的 **More** (更多)，然後選取 **Tutorials** (教學課程)。

![從 Gallery 首頁選取教學課程](media/machine-learning-gallery-tutorials/select-tutorials-in-gallery.png)

 **[Tutorials](https://gallery.cortanaintelligence.com/tutorials)**
 (教學課程) 頁面上會顯示最近新增與最受歡迎的教學課程清單。
按一下 **See all** (查看全部) 即可檢視所有的教學課程。
在這個頁面上，您可以瀏覽 Gallery 中所有的教學課程，或者選取頁面左側的篩選條件並在頂端輸入搜尋字詞進行搜尋。

 按一下任何一個教學課程即可開啟教學課程的詳細資料頁面，並讀取更多有關該教學課程的資訊。
在此頁面上，您可以留言、提供意見反應，或透過留言區段提問。 您甚至可以使用 LinkedIn 或 Twitter 的分享功能，與朋友或同事分享該教學課程。 您也可以用電子郵件寄出教學課程連結，以邀請其他使用者檢視該頁面。

![與朋友分享此項目](media/machine-learning-gallery-how-to-use-contribute-publish/share-links.png)

![新增您自己的留言](media/machine-learning-gallery-how-to-use-contribute-publish/comments.png)

## <a name="contribute"></a>參與
您可以向 Gallery 投搞您自己的教學課程，來幫助其他使用者解決問題或學習概念。

### <a name="create-a-tutorial"></a>建立教學課程
遵循下列步驟建立教學課程並向 Cortana Intelligence Gallery 投稿：

* 使用您的 Microsoft 帳戶登入資源庫
* 按一下您在視窗頂端的影像，然後按一下您的名稱
  
    ![按一下您的帳戶名稱](media/machine-learning-gallery-tutorials/click-account-name.png)
* 按一下 **New Item** (新增項目)
  
    ![按一下 [New Item] (新增項目)](media/machine-learning-gallery-collections/click-new-item.png)
* 對於 **Item Type** (項目類型) 請選取 **Collection** (集合)，然後為集合命名，並提供簡短摘要、說明，以及任何可幫助使用者找到該集合的標籤
  
    ![輸入新集合的資訊](media/machine-learning-gallery-tutorials/create-tutorial-page-1.png)
* 按一下 **Next** (下一步) - 您可以上傳影像檔，或是選取內建影像，影像將會與該集合一起顯示；選擇可以幫助使用者識別集合內容與用途的項目
  
    ![輸入新集合的資訊](media/machine-learning-gallery-tutorials/create-tutorial-page-2.png)
* 按一下 **Next** (下一步) - 您可以決定該教學課程為 **Public** (公開) (任何人都能檢視) 或 **Unlisted** (不列示) (僅具有直接連結的人能夠檢視該教學課程)
  
  > [!IMPORTANT]
  > 一旦您將教學課程設為 **Public** (公開) 後，便無法將它設為 **Unlisted** (不列示)。
  > 
  > 
  
    ![選取 Public (公開) 或 Unlisted (不列示)](media/machine-learning-gallery-tutorials/create-tutorial-page-3.png)
* 按一下 [建立] 

您的教學課程現在是 Cortana Intelligence Gallery 的一部分。 該教學課程將列在您的帳戶頁面上，**Items** (項目) 索引標籤下。

**[前往 GALLERY >>](http://gallery.cortanaintelligence.com)**
