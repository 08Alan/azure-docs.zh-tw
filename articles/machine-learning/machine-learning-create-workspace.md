# 建立 Azure Machine Learning 工作區 
Azure Machine Learning 工作區 (ML 工作區) 包含在管理及發行實驗時所需的工具。如需有關 Machine Learning 工作區的詳細資訊，請參閱連結。
## 建立工作區 ##
1. 登入 Microsoft Azure 帳戶。
2. 在 Microsoft Azure 服務面板中，按一下 **MACHINE LEARNING**。

![][1]
 

1. 按一下 [**建立 ML 工作區**]。
2. 在 [**快速建立**] 頁面上：
	1. 輸入您的工作區資訊。    
	> [AZURE.NOTE] **工作區的擁有者**是您的 Microsoft 帳戶名稱 (name@outlook.com) 或您的組織帳戶名稱。
	2. 按一下 [**建立 ML 工作區**]。
3. Machine Learning 工作區建立後，您會看見它列在 **Machine Learning** 頁面上。
 
![][2]
