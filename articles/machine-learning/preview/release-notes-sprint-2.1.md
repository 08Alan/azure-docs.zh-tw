# <a name="sprint-2-qfe---december-2017"></a>Sprint 2 QFE - 2017 年 12 月 

#### <a name="version-number-01171115323"></a>版本號碼： 0.1.1711.15323

>此處提供如何[尋找版本號碼](https://docs.microsoft.com/en-us/azure/machine-learning/preview/known-issues-and-troubleshooting-guide)的說明。

歡迎使用 Azure Machine Learning 工作臺 的第三個更新的 QFE （快速修復工程） 版本。 此版本是次要版本。 它解決遙測的幾個問題，並幫助以深入了解如何使用產品的產品小組。 知識可移入未來工作改善產品的經驗。 

此外，有兩個重要的更新：

- 資料準備，所以無法顯示在資料準備封裝中的時間序列偵測器中修正的 bug。
- 在命令列工具中，您不再需要為機器學習計算 ACS 叢集佈建 Azure 訂用帳戶擁有者。
