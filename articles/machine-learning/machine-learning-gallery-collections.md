# <a name="discover-and-share-collections-in-the-cortana-intelligence-gallery"></a>探討與分享 Cortana Intelligence Gallery 中的集合
[!INCLUDE [machine-learning-gallery-item-selector](../../includes/machine-learning-gallery-item-selector.md)]

## <a name="collections"></a>集合
**[集合](https://gallery.cortanaintelligence.com/collections)**可讓您將用於處理特定解決方案或概念的實驗、API 和其他資源庫項目群組在一起。 這些可以群組在一起，以方便後續的參考、使用或分享。

## <a name="discover"></a>探索
若要瀏覽 Gallery 中的集合，請開啟 [Gallery](http://gallery.cortanaintelligence.com)，將滑鼠指向 Gallery 首頁頂端的 **More** (更多)，然後選取 **Collections** (集合)。

![從 Gallery 首頁選取集合](media/machine-learning-gallery-collections/select-collections-in-gallery.png)

 **[Collections](https://gallery.cortanaintelligence.com/collections)**
 (集合) 頁面上會顯示最近新增與最受歡迎的教學課程清單。
按一下 **See all** (查看全部) 即可檢視所有的集合。
您可以從此頁面瀏覽 Gallery 中的所有集合。 您也可以選取頁面左側的篩選條件，然後在頂端輸入搜尋字詞來進行搜尋。

 按一下任何一個集合即可開啟集合的詳細資料頁面，並讀取更多有關該集合的資訊。
在此頁面上，您可以留言、提供意見反應，或透過留言區段提問。 您甚至可以使用 LinkedIn 或 Twitter 的分享功能，與朋友或同事分享該集合。 您也可以用電子郵件寄出實驗連結，以邀請其他使用者檢視該頁面。

![與朋友分享此項目](media/machine-learning-gallery-how-to-use-contribute-publish/share-links.png)

![新增您自己的留言](media/machine-learning-gallery-how-to-use-contribute-publish/comments.png)

## <a name="contribute"></a>參與
您可以建立一個集合，以包含您所擁有的項目或已由社群發表的項目。 它們可以是滿足特定解決方案或概念的 Gallery 項目

例如，您可以使用集合將特定主題的項目群組在一起，或者您可以將解決複雜問題的多步驟實驗群組在一起。
Microsoft 發表的初始集合包含用來解決現實世界問題的多步驟機器學習實驗範本，像是線上詐騙偵測、文字分類、零售預測和預測維護。

### <a name="create-a-collection"></a>建立集合
遵循下列步驟建立集合並發表至 Cortana Intelligence 資源庫：

1. 使用您的 Microsoft 帳戶登入資源庫

2.  按一下您在視窗頂端的影像，然後按一下您的名稱
  
    ![按一下您的帳戶名稱](media/machine-learning-gallery-collections/click-account-name.png)

3. 按一下 **New Item** (新增項目)
   
    ![按一下 [New Item] (新增項目)](media/machine-learning-gallery-collections/click-new-item.png)
4. 對於 **Item Type** (項目類型) 請選取 **Collection** (集合)，然後為集合命名，並提供簡短摘要、說明，以及任何可幫助使用者找到該集合的標籤
   
    ![輸入新集合的資訊](media/machine-learning-gallery-collections/create-collection-page-1.png)
5. 按一下 **Next** (下一步) - 您可以上傳影像檔，或是選取內建影像，影像將會與該集合一起顯示；選擇可以幫助使用者識別集合內容與用途的項目
   
    ![輸入新集合的資訊](media/machine-learning-gallery-collections/create-collection-page-2.png)
6. 按一下 **Next** (下一步) - 您可以決定該集合為 **Public** (公開) (任何人都能檢視) 或 **Unlisted** (不列示) (僅具有直接連結的人能夠檢視該集合)
   
   > [!IMPORTANT]
   > 一旦您將集合設為 **Public** (公開) 後，便無法將它設為 **Unlisted** (不列示)。
   > 
   > 
   
    ![選取 Public (公開) 或 Unlisted (不列示)](media/machine-learning-gallery-collections/create-collection-page-3.png)
7. 按一下 [建立] 

您的集合現在是 Cortana Intelligence 資源庫的一部分。 該集合將列在您的帳戶頁面上，**Collection** (集合) 索引標籤下。

### <a name="add-items-to-a-collection"></a>在集合中新增項目
您也可以藉由開啟集合、按一下 **Edit** (編輯)、然後按一下 **Add Item** (新增項目)，以新增項目至您的集合。

![從詳細資料頁面新增項目到集合](media/machine-learning-gallery-collections/add-to-collection-from-details-page.png)

您會看到您已提供到 Gallery 的項目清單，您也可以搜尋 Gallery 來尋找要新增的項目。 按一下以選取項目。 您選取的每個項目都會包含在要新增的項目集中 - **Add** (新增) 按鈕會指示已選取的項目數。

![新增到集合](media/machine-learning-gallery-collections/add-to-collection.png)

或者如果您在瀏覽您想要包含的資源庫時發現項目，您只需要開啟項目、按一下 [新增至集合] ，並指定您想要將它加入的目標集合。

![從詳細資料頁面新增項目到集合](media/machine-learning-gallery-collections/add-to-collection-from-item-details.png)

您可以藉由開啟集合並按一下 [編輯] ，以變更集合的摘要、描述或標記。
當您編輯您的集合時，您也可以使用項目旁邊的箭號按鈕在清單中移動項目，即可變更集合中項目的順序。 而且您可以藉由按一下項目的右上角並選取 [新增/編輯附註] ，即可將附註新增至集合中的項目。 若要從集合中移除項目，請選取 **Remove** (移除)。

![變更集合的詳細資料](media/machine-learning-gallery-collections/change-collection-details.png)
