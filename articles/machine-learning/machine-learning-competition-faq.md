# Microsoft Cortana Analytics Suite 競賽常見問題集

**我可以在哪兒詢問有關資料科學的問題？**

請利用我們的 [Microsoft Azure Machine Learning 論壇](https://social.msdn.microsoft.com/forums/azure/home?forum=MachineLearning)。

**我要如何參加競賽？**

在 [Microsoft Azure Machine Learning](https://studio.azureml.net/?selectAccess=true&o=2%22%20\t%20%22_blank) 中建立免費或付費帳戶。遵循競賽教學課程，並觀看簡介影片。

**我需要是資料科學家才能參加嗎？**

不需要。事實上，我們鼓勵資料愛好者、對資料科學家感到好奇的人，以及其他非專業資料科學家參加這場比賽。我們設計了一些支援文件，讓每個人都能夠參加比賽。

**這需要多久時間？**

您只需要大約 10 - 15 分鐘，便可以參加我們的競賽，並建立資料模型。根據您對機器學習的熟悉度，調整預測實驗可能需要超過 15 分鐘。

**如何計算我的分數？ 競賽進行中與結束後的計算方式有不同嗎？ 什麼是公開和私人分數？**

1.  整個資料集會利用分層隨機分割為訓練 (60%) 和測試 (剩餘的 40%) 資料。隨機分割會以地區 + 區段 + 子群組分層，以確保訓練和測試兩者中的標籤分佈在每個區域中皆一致。  

2.  訓練資料已上傳，並提供給您做為讀取器模組設定中「起始實驗」的一部分。

3.  測試資料會使用相同的分層 (公開測試為 60%，私人測試為 40%)，進一步劃分為公開和私人測試資料。

4.  公開測試資料會在評分的初始回合中使用。結果就是公開分數，而它就是您在提交參賽時於提交歷程記錄中取得的項目。這會在每次您提交參賽時完成。此公開分數是在公開排行榜將您排名時使用。

5.  私人測試資料會在競賽結束後的最終回合中使用。這稱為私人分數。

6.  針對每位參與者，會自動選取擁有最高公開分數的 5 次參賽來計算私人分數。會選取擁有最高私人分數的參賽來參加最終排名，而這最後會決定獎品的得獎者。

**這需要收費嗎？**

不需要，您可以在 Microsoft Azure Machine Learning 中設定免費的工作區來比賽。請記住，來賓帳戶無法儲存資料，因此請確定您在免費的 8 小時到期之前，提交您的參賽。

**如果我贏得競賽，會發生什麼事？**

Microsoft 將會驗證私人排行榜的結果，然後我們會與您連絡。請確定您使用者設定檔中的電子郵件地址為最新狀態。

**如果我贏得競賽，要如何取得獎金？**

如果您是競賽的優勝者，您必須簽署一份資格、授權及發行的宣告。這份表單會重述「競賽規則」。如果得獎者不是美國納稅人，則必須填寫美國稅務表單 W-9，或表單 W-8BEN。
