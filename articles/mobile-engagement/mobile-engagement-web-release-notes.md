#Azure Mobile Engagement Web SDK 版本資訊

##2\.0.1 (06/10/2016)

-   SDK 已在 IE8 和 IE9 上停用。
-   已修正 Opera 網頁瀏覽器偵測。

##2\.0.0 (06/07/2016)

-   Azure Mobile Engagement 的最初發行版本。
-   appId 組態取代為連接字串組態。
-   已新增 API 以啟用/停用代理程式。
-   增強安全性。
-   使用來自瀏覽器的原生 JSON API。
-   已移除在裝置之間傳送與接收訊息的 API。
-   已移除和已被取代的 XMPP 連線相關的回呼。 
-   此版本不支援 Reach 功能。
