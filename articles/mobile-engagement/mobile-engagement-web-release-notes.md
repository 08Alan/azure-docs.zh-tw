#Azure Mobile Engagement Web SDK 版本資訊

##版本 2.0.1 (2016/6/10)

-   在 Internet Explorer 8 和 Internet Explorer 9 中停用 SDK。
-   已修正 Opera 網頁瀏覽器偵測。

##版本 2.0.0 (2016/6/7)

-   發行 Azure Mobile Engagement。
-   更新後，連接字串組態將能取代 `appId` 組態。
-   已新增 API 以啟用及停用代理程式。
-   增強安全性。
-   更新後，可使用來自瀏覽器的原生 JSON API。
-   移除在裝置之間傳送和接收訊息的 API。
-   已移除和已被取代的 XMPP 連線相關的回呼。
-   移除觸達功能支援。
