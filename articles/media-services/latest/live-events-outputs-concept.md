# <a name="live-events-and-live-outputs"></a>即時事件與即時輸出

Azure 媒體服務可讓您在 Azure 雲端上將實況活動傳遞給客戶。 若要在媒體服務 v3 設定即時串流事件，您必須了解[即時事件](https://docs.microsoft.com/rest/api/media/liveevents)和[即時輸出](https://docs.microsoft.com/rest/api/media/liveoutputs)。
