# <a name="common-access-panel-tasks"></a>常見的存取面板工作

存取面板是一個 Web 入口網站，可讓您︰

- 管理及啟動以雲端為基礎的應用程式
- 管理群組成員資格
- 管理設定檔的設定 

![存取面板][1] 

本主題可提供您有關執行常見工作和針對問題進行疑難排解的指示。



## <a name="how-to-troubleshoot-accessing-an-app"></a>如何針對應用程式存取進行疑難排解

1.  確認您的瀏覽器是否已正確安裝存取面板擴充功能。  
請依照[下列步驟](active-directory-saas-ie-troubleshooting.md)來針對 IE 的擴充功能進行疑難排解。
2.  確定瀏覽器不是處於私人或匿名模式。
3.  如果仍無法存取應用程式，請連絡系統管理員或技術服務人員以獲得進一步的協助。

## <a name="how-to-change-your-password"></a>如何變更密碼

1.  在 [存取面板] 中，按一下右上方的使用者控制項，然後按一下 [設定檔]。
2.  在 [管理帳戶] 底下，按一下 [變更密碼]。
3.  輸入您的舊密碼，然後輸入新密碼並加以確認。
4.  按一下 [提交] 。

如需詳細資訊，請參閱[如何更新自己的密碼](active-directory-passwords-update-your-own-password.md)。


## <a name="how-to-setup-password-reset"></a>如何設定密碼重設

1.  在 [存取面板] 中，按一下右上方的使用者控制項，然後按一下 [設定檔]。
2.  在 [管理帳戶] 底下，按一下 [設定自助密碼重設]。
3.  依照對話方塊中的步驟進行並提交。

如需詳細資訊，請參閱[如何重設密碼](active-directory-passwords-update-your-own-password.md#how-to-reset-your-password)。


## <a name="how-to-add-new-apps-or-request-permission-to-app-access"></a>如何新增應用程式或要求應用程式的存取權限

1.  在 [存取面板] 中，按一下 [新增應用程式]。
2.  搜尋或尋找您要新增的應用程式，然後按一下 [新增]。  
如果應用程式需要權限，您可能需要等待系統管理員核准。

如果找不到您要尋找的應用程式，請連絡系統管理員以獲得進一步協助。


## <a name="how-to-manage-your-group-memberships"></a>如何管理群組成員資格

1. 在 [存取面板] 中，按一下 [群組] 應用程式圖格。
2. 若要建立群組，請在 [我擁有的群組] 底下，按一下 [建立群組]，然後依照指示進行。
3. 若要加入群組，請在 [我所在的群組] 底下，按一下 [加入群組]，然後依照指示進行。

**備註**：
  
- 按一下您所隸屬的群組，可讓您檢視詳細資料及脫離群組。
- 按一下您所擁有的群組，則可讓您檢視詳細資料、新增或移除成員，以及脫離群組。
