# 開始使用 Azure AD Privileged Identity Management

## 啟動特殊權限身分識別管理擴充功能

1. 在 Azure Active Directory 中建立目錄。
2. 將使用者加入至新目錄，包括幾個全域管理員。

    >[AZURE.NOTE] 確定您以使用者的公司或學校帳戶來設定使用者。Microsoft 帳戶無法目前註冊 Azure AD PIM。

3. 以該目錄的全域管理員身分登入 [Azure 入口網站](https://portal.azure.com/)。
4. 按一下 Azure 入口網站右上角的使用者名稱。
5. 選取您剛建立的目錄。
6. 按一下左側導覽列中的 [新增] 圖示。
7. 從 [建立] 功能表中選取 [安全性 + 身分識別]。
8. 選取 [Azure AD Privileged Identity Management]。
9. 保留勾選 [釘選到儀表板] 核取方塊，然後按一下 [建立] 按鈕。[特殊權限身分識別管理] 儀表板隨即開啟。
