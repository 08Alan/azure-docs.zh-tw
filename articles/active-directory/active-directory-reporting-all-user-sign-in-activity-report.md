# Azure Active Directory 所有使用者登入活動報告

*這份文件是 [Azure Active Directory 報告指南](active-directory-reporting-guide.md)的一部分。*

 >[AZURE.IMPORTANT]> - 所有使用者登入活動報告是私人預覽中的高階報告。若要取得報告的存取權限，請使用您的網域名稱 (例如 **contoso.onmicrosoft.com**) 傳送電子郵件到 aadreportinghelp@microsoft.com


「所有使用者登入活動」報告包含您目錄中來自所有使用者的所有登入，包括日期與時間、UPN、IP 位址，和使用者登入的應用程式。


 >[AZURE.IMPORTANT]> - 所有使用者登入活動報告目前只能透過報告 API 提供。如需詳細資訊，請參閱[開始使用報告 API](active-directory-reporting-api-getting-started.md)。
