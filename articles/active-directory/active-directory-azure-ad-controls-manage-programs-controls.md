# <a name="manage-programs-and-their-controls"></a>管理程式及其控制項 

Azure AD 包括群組成員和應用程式存取的存取權檢閱，也就是確保監督誰可以存取的控制項範例。 這些控制項可讓組織有效地處理其治理、風險管理和合規性需求。  

## <a name="how-to-manage-programs-and-their-controls"></a>如何管理程式及其控制項
您可以簡化追蹤和收集不同用途的存取權檢閱，方法是將它們組織到程式中。  可將每個存取權檢閱連結至程式中，如此在為稽核員準備報告時，只會看到特定方案的範圍內存取權檢閱。

若要查看程式清單，請移至[存取權檢閱頁面](https://portal.azure.com/#blade/Microsoft_AAD_ERM/DashboardBlade/)並變更為 [程式集] 索引標籤。

一律會出現一個「預設程式」程式。  如果您是全域系統管理員角色，您可以建立其他程式。  比方說，您可以針對每個合規性方案或業務目標建立一個程式。

如果您不再需要程式，且該程式未連結任何控制項，您可以刪除它。
