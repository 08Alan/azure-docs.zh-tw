# 如何開始使用 Azure AD

[什麼是 Azure AD？](active-directory-whatis.md)<br> [運作方式](active-directory-works.md)<br> [後續步驟](active-directory-next-steps.md)<br> [深入了解](active-directory-learn-map.md)


準備好開始了嗎？ Azure AD 免費提供給所有人。立即註冊 Azure 試用版訂用帳戶，您的目錄將會等著您。目錄就屬於您，而不論您是否另購任何產品。[立即試用 Azure AD](http://azure.microsoft.com/trial/get-started-active-directory/)

## 我應該如何決定測試、暫存、生產要使用的目錄？

如果您有現有的 Microsoft 雲端服務 (如 Office 365) 訂用帳戶，然後在 Azure AD 中已經有一個目錄，而且該目錄也可以使用。該目錄已驗證登入 Office 365 的使用者。您只需要使用您用來登入 Office 365 的相同工作或學校帳戶來註冊 Azure 即可。然後您可以管理 Azure 管理入口網站中的目錄，並視需要新增用於測試的目錄，依此類推。[深入了解如何在 Azure 中管理 Office 365 訂用帳戶的目錄](active-directory-how-subscriptions-associated-directory.md#manage-the-directory-for-your-office-365-subscription-in-azure)。

如果您還沒有 Microsoft 雲端服務的訂用帳戶，則當您註冊 Azure 時，系統會為您的工作或學校佈建新的目錄，而且您可以使用該目錄做為您的主要目錄。


## 如何新增專屬的註冊網域名稱？
想要新增自訂網域名稱，或使用您專屬的註冊網域名稱嗎？ [深入了解新增自訂網域](active-directory-add-domain.md)。

## 我要如何加上公司商標？
有了 Azure AD Premium，您可以在 Azure 登入頁面和存取面板頁面加上公司標誌和其他商標。[在登入和存取面板頁面加上公司商標](active-directory-add-company-branding.md)

## 我要如何將 Azure AD 與另一個目錄整合？
您可以設定內部部署目錄與 Azure AD 之間的目錄同步處理 (包括密碼同步處理)，並啟用存取內部部署和雲端中應用程式的單一登入。[深入了解目錄整合](active-directory-aadconnect.md)。

## 我可以如何建立及編輯使用者？
為存取 Microsoft 雲端服務的每位使用者建立帳戶。根據預設，使用者沒有系統管理員權限，但是您可以指派給他們。[深入了解如何建立 Azure AD 使用者](active-directory-create-users.md)。

## 檢視存取和使用情況報告
Azure AD 為系統管理員提供了幾種報告。在 Azure AD 的付費版本中有更多種報告，而且還有一個可讓您建立專屬自訂報告的報告 API。[深入了解 Azure AD 報告](active-directory-view-access-usage-reports.md)。
