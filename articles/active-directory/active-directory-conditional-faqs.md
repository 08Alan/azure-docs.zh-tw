# 條件式存取常見問題集

**如果我使用 Office 365 行動裝置管理，我是否可以在 Azure AD 中使用裝置條件？**
 
使用 O365 MDM 時，所有裝置原則會都隨 Office 365 一起維護。如果您使用 O365 MDM，為了避免 Office 365 中設定的原則與 Azure AD 中設定的原則發生衝突，我們目前並不允許在 Azure AD 中設定這些原則。
