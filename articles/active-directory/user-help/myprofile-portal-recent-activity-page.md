# <a name="view-your-recent-sign-in-activity-on-the-my-profile-preview-portal"></a>在我的設定檔 (預覽) 入口網站檢視您最近的登入活動
您可以使用您的公司或學校帳戶，從 [我的設定檔] 入口網站的 [最近的活動] 頁面檢視所有您最近的登入活動。

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-preview-notice-myprofile.md)]

## <a name="view-your-recent-sign-in-activity"></a>檢視您最近的登入活動
1. 登入您的公司或學校帳戶，然後移至您的 https://myprofile.microsoft.com/ 頁面。

2. 從左瀏覽窗格中選取 [最近的活動]，或從 [最近的活動] 區塊選取 [檢閱活動]。

    ![[我的設定檔] 頁面，其中 [最近的活動] 連結已醒目提示](media/myprofile-portal/myprofile-portal-recent-activity-links.png)

3. 展開並檢閱每個登入項目，以確定您可識別出每個項目。 如果您發現不熟悉的登入項目，我們強烈建議您變更密碼，以協助保護您的帳戶 (如果已遭到入侵)。

    ![[最近的活動] 頁面中的登入詳細資料已展開](media/myprofile-portal/myprofile-portal-recent-activity-page.png)

    >[!Note]
    >如果您認為您的帳戶已遭到入侵，請參閱[如何存取遭盜用的 Microsoft 帳戶](https://support.microsoft.com/help/10494/microsoft-account-how-to-access-a-compromised-account)，以取得重設密碼或將您帳戶解除封鎖的更多協助。

## <a name="next-steps"></a>後續步驟

- 檢視最近的登入活動之後，您可以選取要檢視或管理您的[安全性資訊](user-help-security-info-overview.md)、連線的[裝置](myprofile-portal-devices-page.md)或[組織](myprofile-portal-organizations-page.md)，或檢視您的組織如何使用[您的資料](myprofile-portal-privacy-page.md)。
