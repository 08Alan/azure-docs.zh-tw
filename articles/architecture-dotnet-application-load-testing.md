# Azure 中的負載測試

雲端型負載測試服務可以對您的 Web 應用程式進行負載測試。使用 Visual Studio 可以建立負載測試來判定 Web 應用程式的容量和效能；不需要使用您專屬的 Azure VM。您可以改為讓[雲端式負載測試服務](http://tfs.visualstudio.com/zh-tw/learn/load-testing)為您建立這些測試。
