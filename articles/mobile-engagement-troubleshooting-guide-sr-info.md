# 服務要求資訊的疑難排解指南

當您開啟任何 Azure Mobile Engagement 服務要求時，請提供下列資訊：
 
## 識別碼：請提供任何與您的問題有關的適當識別碼
- 應用程式識別碼
- 活動識別碼
- 裝置識別碼
- 使用者識別碼
- 使用者名稱
- 應用程式資訊標記
 
## 錯誤：請提供任何與您的問題有關的適當錯誤資訊
- 發生問題的 API 或 UI 區段名稱
- 收到的任何錯誤訊息文字
- 從[疑難排解指南](http://go.microsoft.com/fwlink/?LinkId=524382)中已執行的任何測試結果


## 程式碼：請提供任何與您的問題有關的適當編碼資訊
- 應用程式的 SDK 版本與平台 (Android SDK 2.4.1、 iOS 1.16.2 等等)
- 應用程式產品的下載位置 (或是您開發的 APK/TGZ 檔案)
- 「AndroidManifest.xm」和/或任何您的應用程式與 Azure Mobile Engagement 相關的程式碼片段 (進階疑難排解)
