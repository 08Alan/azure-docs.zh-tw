追蹤網站的工作
==============

在本主題中，您將了解如何使用工作項目追蹤工具來管理您的網站工作。

作法：將工作項目新增至待處理項目
--------------------------------

1.  瀏覽到您的小組專案 blade。

2.  在底部的 [工作] 鏡頭內，按一下待處理項目部分。

    此時便會啟動待處理項目 blade。您的待處理項目會包含已排定優先順序的網站工作清單。

    ![Open backlog](./media/visual-studio-online-work-item-tracking/visual-studio-online-open-backlog.png)

3.  在頂端的文字方塊中輸入標題並按 Enter 鍵，以新增項目。

    如果使用 Scrum 流程範本，您可以選擇新增錯誤或產品待處理項目，方法是從右側的下拉式清單中進行選取。

4.  按一下工作項目來開啟並新增其他詳細資料。

    ![Open work item](./media/visual-studio-online-work-item-tracking/visual-studio-online-open-work-item.png)

5.  在工作項目表單上，您可以編輯標題、新增說明、將工作項目指派給使用者，及將其標示為完成。

6.  在工作項目表單底部，您可以為工作項目加上註解並檢視其歷程記錄。

    若要探究工作項目的歷程記錄，您可以按一下變更，則包含更多發生情形詳細資料的 blade 便會隨即開啟。

作法：管理待處理項目
--------------------

保存已排定好優先順序的待處理項目是順利管理網站工作的一大重點。這包含兩個部分：

### 在工作完成時將其移除

1.  開啟您要標示為完成的工作項目。
2.  勾選標示為 [已完成] 的核取方塊。
3.  儲存此工作項目。
4.  待處理項目上會保留此項目的虛設常式，直到您重新整理待處理項目，或關閉並重新開啟待處理項目為止。

![Completed work item](./media/visual-studio-online-work-item-tracking/visual-studio-online-completed-work-item.png)

### 按優先順序保存待處理項目

1.  將滑鼠移至您要變更其優先順序的工作項目。
2.  按一下工作項目左側上的移駐夾列。
3.  在待處理項目上，將工作項目拖放至新位置。

![Dragging work item](./media/visual-studio-online-work-item-tracking/visual-studio-online-dragging-work-item.png)

作法：檢視最近完成的工作
------------------------

1.  在小組專案 blade 上，按一下已完成項目 blade。
2.  檢視最近完成的工作清單。

![Dragging work item](./media/visual-studio-online-work-item-tracking/visual-studio-online-completed-items.png)
