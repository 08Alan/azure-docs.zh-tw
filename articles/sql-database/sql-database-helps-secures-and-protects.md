# Azure SQL Database 有助於安全防護

## 確保您的應用程式資料安全無虞

Azure SQL Database 在雲端中支援數百萬的關鍵任務資料庫，這些資料庫都受益於高可用性、安全性、相容性和高效能。從最小的創業公司到最大的企業，資料安全性逐漸成為所有企業最在意的項目，而 SQL Database 支援各種規模的企業。

![helps-secures-and-protects](./media/sql-database-helps-secures-and-protects/sql-database-helps-secures-and-protects.png)

## 若有緊急狀況

您只能希望不需要存取昨天的備份，或從災害中回復。但是，如果您發現自己身處在此情況下，SQL Database 可以幫助您。透過 99.99% 可用性、自動備份、還原時間點、內建異地複寫和容錯移轉選項，您可以放心確保應用程式安全無虞。

[閱讀文件](sql-database-business-continuity.md)

## 最大的相容性組合

SQL Database 在產業中具有最大的相容性組合。因此，如果您需要 HIPPA、FERPA 或甚至新加坡 MTCS 層級 3，請不用擔心，我們會提供支援。

[閱讀文件](https://www.microsoft.com/TrustCenter/Compliance/default.aspx)

## 保護層

SQL Database 提供多個資料保護層。您的資料會在靜止時、移動時或使用中時受到加密資料的保護，只針對資料庫或應用程式驗證獲授權的使用者，並限制使用者對適當資料子集的存取。此外，SQL Database 提供持續的監視和稽核活動，協助偵測潛在的威脅，並提供發生漏洞時的重要事件記錄。這些功能可透過快速實作功能達到平衡，並且能降低安全性風險，而不會危及開發人員生產力或客戶的體驗。
