# <a name="xml-processing"></a>XML 處理
使用企業整合套件搭配 Azure Logic Apps，可以更輕鬆地驗證和處理 B2B 案例中您與合作夥伴交換的 XML 文件。 以下是您可以處理 XML 訊息的方法：

* [XML 驗證](logic-apps-enterprise-integration-xml-validation.md "了解 XML 訊息驗證") - 根據特定的結構描述，驗證來自來源端點的訊息。
* [XML 轉換](../logic-apps/logic-apps-enterprise-integration-transform.md "了解 XML 訊息轉換和對應") - 根據目的地端點的需求轉換 XML 訊息。
* [一般檔案編碼與一般檔案解碼](logic-apps-enterprise-integration-flatfile.md "了解一般檔案編碼/解碼") - 編碼或解碼一般檔案。 
* [XPath](https://msdn.microsoft.com/library/mt643789.aspx) - 可擴充訊息，並從訊息擷取特定的屬性。 然後您可以使用擷取的屬性將訊息傳送至目的地或中繼端點。

## <a name="try-out-xml-processing"></a>試用 XML 處理
使用 Azure Logic Apps 中的 XML 功能，[部署全功能的邏輯應用程式](https://github.com/Azure/azure-quickstart-templates/tree/master/201-logic-app-veter-pipeline)。
