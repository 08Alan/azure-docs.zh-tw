# <a name="logic-apps-documentation"></a>Logic Apps 文件

Logic Apps 是設計工具，將處理程序模型化和自動化至工作流程。 雲端和內部部署中有 許多連接器可快速整合各項服務和通訊協定。 邏輯應用程式是以觸發程序為開端 (如「當帳戶加入至 Dynamics CRM 時」)，而在觸發後可以開始處理各種組合的動作、轉換和條件邏輯。
