# Operational Insights 搜尋
Microsoft Azure Operational Insights 的核心是我們的搜尋功能，可讓您結合和相互關聯您環境內多個來源的任何電腦資料。智慧套件也是由搜尋發動，可讓您針對特定問題領域進行度量。


## 搜尋資料
您可以在 Azure Operational Insights 中使用 [**搜尋**] 功能，尋找您正在搜尋的資料 

![搜尋磚的影像](./media/operational-insights-search/overview-search.png)

在 [**搜尋**] 頁面上，您可以建立查詢，然後在搜尋時，您可以使用 Facet 控制篩選結果。您也可以建立進階查詢來轉換、篩選和報告結果。

![搜尋頁面的影像](./media/operational-insights-search/search-01.png)

常見的搜尋查詢會出現在大部分搜尋頁面中。在整個 Operational Insights 主控台中，您可以按一下磚或向內切入其他項目，使用 [**搜尋**] 來檢視項目的詳細資料。
您可以使用 Facet 和篩選控制項以縮小結果。包含自動完成功能的輔助鍵入可協助您快速瞭解語法和產生結果。
