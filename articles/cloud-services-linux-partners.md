# Azure Linux 的夥伴解決方案

請連絡此頁面上的夥伴，以協助您在 Azure 中提供 Linux 架構的雲端解決方案。

## Bitnami

[![Bitnami][![Bitnami]][]

<http://bitnami.org/azure/get-started>

BitNami 為熱門的 Web 應用程式提供完整設定的映像，只要按一下即可部署到 Azure。Azure 的 BitNami WebPack 和 DevPack 映像提供完整設定且可立即執行的環境，免費供 Web 和應用程式開發人員使用。BitNami WebPack 隨附 Wordpress、Drupal 和 Joomla!，讓開發人員能夠選擇最佳平台來開發網站和部落格。BitNami DevPack 提供的開發環境備有齊全的 Ruby on Rails、PHP、Django 和 Java 語言執行階段及其所有相依性，開發人員在雲端中建立 Web 應用程式所需的一切盡在其中。

## Opscode Chef

[![Opscode][![Opscode]][]

[www.opscode.com/partners/microsoft][![Opscode][![Opscode]]

Opscode 是 Chef 幕後的公司，Chef 是快速成長的雲端基礎結構自動化架構，成千上萬的組織都已採用，從小型的創業者到財富雜誌 100 大企業都有。Chef 有靈活且可重複使用的範本 (「食譜」)，可讓您輕鬆地管理、調整和重建 Azure 環境。

## RightScale

[![Rightscale][![Rightscale]][![Opscode][![Opscode]]

<http://www.opscode.com/partners/microsoft>

RightScale® Inc., 雲端管理可讓組織跨公用、私人和混合式雲端，輕鬆部署和管理業務關鍵應用程式。RightScale 能夠有效率地設定、監視、自動化及控管雲端運算基礎結構和應用程式。自 2006 年起，頂尖的企業有成千上萬台伺服器都開始採用 RightScale 解決方案，包括 Associated Press、CBS Interactive、Intercontinental Hotels Group、PBS 和 Zynga。

## ScaleXtreme

[![ScaleXtreme][![ScaleXtreme]][]

[www.scalextreme.com/partners-windows-azure][![ScaleXtreme][![ScaleXtreme]]

ScaleXtreme 提供新一代的系統和雲端管理產品，讓客戶利用一套整合的產品來管理 Azure 和內部企業伺服器。ScaleXtreme 以雲端為架構，讓客戶立即擁有企業級的自動化能力，相較於傳統解決方案的成本和複雜性，相當划算。

ScaleXtreme 可讓客戶在 Microsoft Azure 上建立、啟動、監視、修補及管理 Windows 和 Linux 伺服器，以及快速部署和調整伺服器基礎結構。ScaleXtreme 以雲端為架構且可立即部署，讓客戶「立即」擁有系統管理和自動化能力，不需要部署內部部署基礎結構。產品功能包括伺服器範本和佈建、監視、預算與成本控制、修補程式管理，以及豐富的自動化能力。

## SUSE Studio

[![SUSE][![SUSE]][]

[www.susestudio.com][![SUSE][![SUSE]]

SUSE 可讓您透過 SUSE Studio 的一鍵式部署功能，輕鬆地將 SUSE Linux Enterprise Server 架構的應用程式延伸至 Azure。透過 SUSE Studio，客戶可以迅速部署雲端備妥應用程式，包括已預先設定、整合且完全受支援的企業級 Linux 作業系統，並於 Azure 上自動啟動，無效率的手動程序幾乎一掃而空。

## Ubuntu Juju

[![Juju][![Juju]][]

<http://www.ubuntu.com/cloud/azure>

在確認了雲端運算為未來趨勢，以及了解到雲端採用對於 IT 組織和開發人員的意義之後，我們開發了 Juju，這是一套創新格局的服務協調工具集，可將應用程式相關的知識 (相依性、關聯性和平台組態) 封裝成鎖鏈。

Juju 鏈是一組預先編寫的雲端服務部署指令。目前數量超過 50 個，適用於 Hadoop、PostgreSQL 和 Drupal 等服務。因為任何人都可以提交指令，數量正不斷地在成長中。

Juju 與佈建緊密相關，但與基本的基礎結構層無關。基本平台可以安裝在公用雲端、私人雲端或硬體上，隨時可搭配 Juju 來部署、控制和調整服務。
