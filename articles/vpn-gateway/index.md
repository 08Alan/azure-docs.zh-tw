#<a name="vpn-gateway-documentation"></a>VPN 閘道文件

Azure VPN 閘道是虛擬私人網路 (VPN)，用途在傳送 Azure 虛擬網路與內部部署位置之間的網路流量，以及 Azure 內的之虛擬網路間 (虛擬網路對虛擬網路) 的網路流量。
