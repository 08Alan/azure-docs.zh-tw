# Azure 搜尋服務中的查詢
> [AZURE.SELECTOR]
- [Overview](search-query-overview.md)
- [Search Explorer](search-explorer.md)
- [Fiddler](search-fiddler.md)
- [.NET](search-query-dotnet.md)
- [REST](search-query-rest-api.md)

整合 Azure 搜尋服務來取得內嵌搜尋經驗的自訂解決方案必須包含可建構搜尋查詢並將其傳送至搜尋引擎進行處理的程式碼。

您可以使用 .NET SDK 或 REST API 來撰寫可供執行查詢的方法。若要進行初步測試或探勘，您可以使用 Azure 入口網站中的內建搜尋總管之類的工具來查詢索引，或使用 Fiddler 對搜尋服務發出任何有效要求。

如需示範如何在程式碼中建構查詢的範例和逐步解說清單，請瀏覽[影片、教學課程、示範和程式碼範例](search-video-demo-tutorial-list.md)。我們提供的每個範例幾乎都包含用來查詢搜尋服務的程式碼。
