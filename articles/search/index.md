# <a name="azure-search-documentation"></a>Azure 搜尋服務文件

Azure 搜尋服務是受管理的雲端搜尋服務，協助您在自訂的應用程式中建置功能強大的搜尋經驗，並透過入口網站檢視來監視服務、索引和查詢活動。 主要功能包括涵蓋多語言的可調整全文搜尋、地區搜尋、篩選及多面向導覽、輸入查詢、結果反白及自訂分析。
