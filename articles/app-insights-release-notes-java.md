# Java 適用的 Application Insights SDK 的版本資訊

[使用 Java 適用的 SDK](app-insights-java-get-started.md)

## 0.9.4 版

- 支援從 32 位元 Windows 電腦收集效能計數器。
- 支援使用新的 ```trackDependency``` 方法 API 手動追蹤相依性。
- 能夠藉由將 ```SyntheticSource``` 屬性加入至報告項目，將遙測項目標記為綜合。
