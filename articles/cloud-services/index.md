# <a name="cloud-services-documentation"></a>雲端服務文件

雲端服務是平台即服務 (PaaS) 實作的一個範例。 它能夠支援可調整、可靠且具有操作成本效益的應用程式。 使用雲端服務，您擁有虛擬化的優點，但您也可以控制虛擬機器以設定、修補和安裝軟體。
